# 入口导航

所有接手者从本目录开始。

| 文件 | 用途 |
| --- | --- |
| `研究驾驶舱.md` | 当前研究方向、状态、优先级和下一步 |
| `当前状态.md` | 今日研究库整体状态和接手摘要 |
| `研究路线图.canvas` | 可缩放、可拖拽的 Obsidian 主路线图 |
| `研究路线图.md` | 路线图说明和小型缩略图 |

## 更新规则

- 新增研究方向后，更新 `研究驾驶舱.md` 和 `研究路线图.canvas`。
- Mermaid 图只做 10 个以内节点的小型缩略图，节点多时改用 Canvas 或子 Canvas。
- 方向状态变化后，更新 `研究驾驶舱.md`。
- 产生关键决策后，更新 `当前状态.md`。
- 入口只放摘要和链接，不承载完整实验解释。
