# I20260523-004 尾部风险与波动风险溢价预警

## 一句话描述

从极端下跌、波动率和风险溢价代理变量中识别市场风险状态，作为动量和小市值策略的防守门控。

## 可实现代理变量

| 指标 | 说明 |
| --- | --- |
| 左尾分位收益 | 全市场或股票池 5% 分位日收益 |
| 跌停占比 | 跌停股票数量占可交易股票比例 |
| 行业同步下跌 | 下跌行业占比和行业相关性上升 |
| 波动率跳升 | 指数或组合已实现波动率快速升高 |
| 方差风险溢价 | 有期权数据后再考虑接入 |

## 使用方式

- 风险状态升高时降低动量策略权重。
- 极端尾部风险时暂停小市值进攻。
- 风险回落后分阶段恢复，不做一次性满仓切换。

## 未来函数审计

所有截面统计必须使用当日可交易时点之前的数据。若只在收盘后生成信号，则只能影响下一交易日。

~~~
