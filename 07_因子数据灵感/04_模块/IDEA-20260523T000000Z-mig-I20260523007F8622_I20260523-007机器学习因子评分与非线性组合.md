# I20260523-007 机器学习因子评分与非线性组合

## 一句话描述

用机器学习和非参数方法研究因子之间的非线性、交互项和状态依赖，输出可解释的排序或门控候选。

## 第一阶段边界

| 边界 | 说明 |
| --- | --- |
| 不直接实盘 | 只做研究和解释 |
| 不替代规则 | 输出候选，不覆盖已有透明策略 |
| 必须样本外 | 采用滚动训练和严格时间切分 |
| 必须可解释 | 记录特征贡献、分位单调性和失败样本 |

## 候选任务

- 检查平滑上涨因子在哪些市场状态有效。
- 检查动量、波动率、相关性、尾部风险之间是否存在交互。
- 为小市值策略识别质量阈值，而不是线性加权质量分。

## 风险与限制

机器学习很容易把噪声当规律。任何结果都必须先通过 [[因子稳健性与失效监控]]。
