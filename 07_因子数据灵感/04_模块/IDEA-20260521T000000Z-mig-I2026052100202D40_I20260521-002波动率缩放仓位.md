# I20260521-002 波动率缩放仓位

## 一句话描述

根据组合或市场已实现波动率动态调整总仓位，高波动时降低风险预算，低波动时恢复正常仓位。

## 模块定义

可测试三个版本：

| 版本 | 定义 |
| --- | --- |
| 市场波动率版 | 使用基准指数或 ETF 池等权组合波动率 |
| 持仓波动率版 | 使用当前持仓 ETF 的历史波动率 |
| 组合净值版 | 使用策略自身近期净值波动率 |

仓位缩放公式先用保守版本：

```text
target_exposure = clamp(target_vol / realized_vol, min_exposure, max_exposure)
```

## 适用策略

| 策略 | 适用方式 |
| --- | --- |
| [[ETF双池动量轮动]] | 调整总仓位，不改变 TopN 排名 |
| [[动量趋势日内做T]] | 高波动时降低隔夜或日内暴露 |

## 验证设计草案

| 项目 | 内容 |
| --- | --- |
| 基准策略 | ETF 双池 TopN |
| 实验版本 | 加波动率缩放 |
| 消融方式 | 固定仓位、市场波动率缩放、持仓波动率缩放 |
| 核心指标 | 最大回撤、收益回撤比、换手率、空仓天数 |
