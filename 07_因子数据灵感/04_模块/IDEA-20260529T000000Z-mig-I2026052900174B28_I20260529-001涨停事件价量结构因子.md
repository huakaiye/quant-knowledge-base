# I20260529-001 涨停事件价量结构因子

## 一句话描述

把涨停前的[[价量结构]]转换为可复现的数值因子，用于判断[[涨停事件]]之后是否仍有短线接力收益。

## 来源

来源文献：[[2025_权仓量化_价量因子核心窗口涨停股应用]]

来源观察：聚宽社区文章提出“涨停前核心窗口 + 涨停确认 + 次日竞价买入 + 固定持有两日”的短线事件策略逻辑。

## 双标签

收益来源：[[事件驱动类策略]]、[[横截面动量类策略]]

执行方式：[[股票轮动]]、[[集合竞价]]、[[日频调仓]]

## 模块定义

候选股票必须先满足：

- T 日为涨停日。
- T 日不是一字涨停。
- T 日不属于 ST、停牌、上市过短、极低成交额标的。
- 买入信号只能在 T 日收盘后形成，最早 T+1 执行。

初始因子族：

| 因子族 | 口径草案 | 预期含义 |
| --- | --- | --- |
| 结构强度 | T-5 到 T-1 累计收益、T-1 收盘位置 | 涨停前已有资金关注 |
| 试盘冲高 | T-N 内高点冲击但收盘未过热 | 有资金试探但未完全释放 |
| 缩量整理 | 试盘后成交量低于近 N 日均量 | 卖压收缩或筹码稳定 |
| 波动收敛 | 涨停前 3-10 日振幅下降 | 分歧收敛 |
| 涨停突破 | T 日收盘突破近 N 日高点且封板 | 事件确认 |
| 流动性过滤 | T 日成交额、近 20 日成交额 | 避免不可交易小票 |

## 适用策略

| 策略 | 适用方式 | 备注 |
| --- | --- | --- |
| [[涨停事件价量结构策略]] | 作为候选过滤和排序因子 | 第一阶段只研究，不交易 |
| [[小市值日内做T]] | 作为事件驱动子模块参考 | 仅共享流动性和执行约束，不直接接入 |

## 数据需求

| 数据 | 当前是否具备 | 说明 |
| --- | --- | --- |
| 日线 OHLCV | 待确认 | 用于涨停识别和价量结构 |
| 涨跌停价格 | 待确认 | 判断一字板和不可成交 |
| 分钟线 | 待确认 | 可辅助估算开盘初段流动性 |
| 基本交易状态 | 待确认 | ST、停牌、上市天数过滤 |

## 预期改善

- 收益：若因子有效，应提高涨停事件后 1-3 日收益。
- 回撤：应减少涨停后快速断板和大幅低开的标的。
- 换手：不会降低换手，主要改善交易质量。
- 稳定性：必须通过年份切分和样本外验证，否则不采纳。

## 验证设计草案

| 项目 | 内容 |
| --- | --- |
| 基准策略 | 全部非一字涨停股 T+1 买入，持有 2 日 |
| 实验版本 | 在基准上逐个加入价量结构因子 |
| 样本内 | 待平台恢复后设定 |
| 样本外 | 至少按年份滚动切分 |
| 消融方式 | 单因子、去流动性、去波动收敛、去突破确认、去极端收益样本 |
