# I20260523-009 质量防御与价值质量组合

## 一句话描述

用盈利质量、稳健性、估值和投资保守性构建防御型股票评分，作为小市值和动量策略的过滤或权重倾斜。

## 模块定义

| 子项 | 示例指标 |
| --- | --- |
| 盈利能力 | ROE、毛利率、净利率 |
| 稳健性 | 盈利波动、负债率、现金流质量 |
| 投资保守性 | 资产增长、资本开支扩张 |
| 估值约束 | PE、PB、股息率或行业内估值分位 |

## 使用方式

- 正常状态下作为候选过滤。
- 风险状态升高时提高质量权重，降低纯动量权重。
- 对小市值策略，优先剔除极低质量样本，而不是线性扣分。

## 风险与限制

基本面指标必须使用公告可见时点，不能使用未来财报修订后的数据。

~~~
