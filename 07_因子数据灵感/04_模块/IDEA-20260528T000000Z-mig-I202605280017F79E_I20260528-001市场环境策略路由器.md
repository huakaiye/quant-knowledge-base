# I20260528-001 市场环境策略路由器

## 一句话描述

根据市场环境信号，在小市值、ETF 动量、质量防御、低波/债券/现金等策略之间分配权重或开关，优先降低不适配策略的风险预算。

## 模块边界

策略路由器不是新的选股模型。它只回答：

- 当前哪些策略允许开仓。
- 哪些策略只能低仓位运行。
- 哪些策略需要暂停或切到防守。
- 每个策略的最大风险预算是多少。

## 输入

| 输入 | 示例 | 用途 |
| --- | --- | --- |
| 市场趋势 | 宽基 20/60 日收益、均线位置 | 判断系统方向 |
| 市场广度 | 上涨家数、MA20 上方占比 | 判断普涨还是窄主线 |
| 风格强弱 | 小盘/大盘、成长/价值、红利/成长 | 判断风格奖励方向 |
| 横截面离散 | Top1/Top5 分数、截面收益标准差 | 判断轮动是否有价值 |
| 流动性风险 | 成交额、跌停数、左尾收益、小票成交占比 | 判断小市值和高换手是否可交易 |
| 策略健康度 | 子策略滚动回撤、波动、换手 | 判断策略是否进入失效观察 |

## 输出

| 输出 | 说明 |
| --- | --- |
| `market_state` | 当前市场环境标签 |
| `allowed_strategies` | 允许运行的策略列表 |
| `blocked_strategies` | 暂停运行的策略列表 |
| `strategy_weight_caps` | 各策略最大权重上限 |
| `risk_budget_multiplier` | 总风险预算倍率 |
| `reason_codes` | 本次路由决策的触发原因 |

## 初始路由规则

| 状态 | 小市值 | ETF 动量 | 质量/价值 | 低波/防守 |
| --- | ---: | ---: | ---: | ---: |
| 全面风险偏好上行 | 高 | 中高 | 中 | 低 |
| 窄主线强趋势 | 低 | 高 | 中 | 低 |
| 中高离散轮动 | 中 | 高 | 中 | 低 |
| 低离散震荡 | 低 | 低 | 中 | 中高 |
| 质量/价值修复 | 中，但必须质量过滤 | 中低 | 高 | 中 |
| 流动性压力 | 暂停或极低 | 低 | 中 | 高 |
| 急跌后急反弹 | 低 | 低到中 | 中 | 高 |
| 监管/质量重估 | 只保留质量小市值 | 中低 | 高 | 中高 |

## 第一版实现原则

- 先做“建议日志”，不直接改变交易。
- 第二阶段只做降权，不做全量策略切换。
- 第三阶段才允许策略切换。
- 状态切换必须有确认和冷却期，避免来回切。
- 所有阈值使用训练期固定或滚动窗口，禁止全样本分位数。

## 证伪标准

| 观察 | 决策 |
| --- | --- |
| 路由建议不能解释各策略收益差异 | 暂停路由器，回到单策略状态识别 |
| 路由只在少数极端日有效 | 改成风险预算模块，不做策略切换 |
| 路由收益被换手和延迟吞噬 | 保留状态日志，停止交易层接入 |
| 路由能降低回撤但牺牲过多收益 | 改为只在风险状态触发 |
