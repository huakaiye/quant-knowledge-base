# I20260523-006 特征协方差风险模型

## 一句话描述

把市值、动量、质量、波动率、流动性等特征看作风险暴露线索，用于检查组合是否在无意中押注同一种风险。

## 第一阶段简化版

| 工具 | 定义 |
| --- | --- |
| 因子相关性 | 不同因子分数的截面相关性 |
| 持仓重叠度 | 不同策略或因子选出的股票/ETF 重叠比例 |
| 风险暴露表 | 持仓在市值、行业、波动率、流动性上的暴露 |
| 拥挤提示 | 多个策略同时偏向同一风险时降权 |

## 适用场景

- 小市值策略和低流动性暴露重叠。
- 动量策略买入高波动资产，组合实际变成高波动押注。
- 多策略组合表面分散，实质持仓高度重叠。

## 风险与限制

完整风险模型需要较高数据质量和长期样本，第一阶段只做诊断工具。
