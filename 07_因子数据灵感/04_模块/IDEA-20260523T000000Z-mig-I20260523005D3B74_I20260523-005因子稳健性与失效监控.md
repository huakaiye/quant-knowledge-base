# I20260523-005 因子稳健性与失效监控

## 一句话描述

为每个新因子建立“来源、经济解释、样本外表现、增量贡献、失效场景”的入库和监控流程。

## 检查项

| 检查项 | 说明 |
| --- | --- |
| 文献来源 | 是否来自论文、实务经验或数据观察 |
| 经济解释 | 是否能解释为什么应当赚钱 |
| 样本外 | 是否跨年份、跨市场状态有效 |
| 增量贡献 | 是否只是复制已有因子暴露 |
| 执行成本 | 收益是否能覆盖换手、滑点和冲击 |
| 失效场景 | 在什么市场状态下会失效 |

## 适用方式

- 所有新因子实验进入研究台账前必须完成此检查。
- 表现只在局部年份有效的因子，优先归入状态依赖模块。
- 与已有因子高度相关的因子，不直接升级为新模块。

## 输出物

平台恢复后应建立一张因子治理表，记录每个因子的上线状态和最近一次复核结论。

~~~
