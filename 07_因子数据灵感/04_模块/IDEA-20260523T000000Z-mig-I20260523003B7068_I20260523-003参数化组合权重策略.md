# I20260523-003 参数化组合权重策略

## 一句话描述

用少量可解释参数把资产特征映射为组合权重，让权重随分数连续变化，而不是只买 Top1 或固定 TopN 等权。

## 模块定义

```text
raw_weight_i = base + b1 * momentum_i - b2 * volatility_i + b3 * quality_i
weight_i = normalize(clamp(raw_weight_i, min_weight, max_weight))
```

## 可测试版本

| 版本 | 定义 |
| --- | --- |
| 动量单因子权重 | 分数越高权重越大 |
| 动量加波动惩罚 | 高动量但高波动时不满配 |
| 状态门控权重 | 风险状态高时整体权重压缩 |

## 风险与限制

- 参数不能过多，初期最多 2 到 3 个。
- 必须设置换手约束，否则连续权重会频繁微调。
