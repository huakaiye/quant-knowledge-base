# I20260521-009 投资保守性过滤

## 一句话描述

用资产增长或投资强度过滤扩张过快的股票，作为小市值和质量策略的低频基本面约束。

## 模块定义

优先测试可得性最强的代理指标：

| 指标 | 说明 |
| --- | --- |
| 总资产增长率 | 资产扩张越快，投资越激进 |
| 固定资产或资本开支增长 | 若数据可得，用于补充投资强度 |
| 负债增长 | 辅助识别高杠杆扩张 |

## 适用策略

| 策略 | 适用方式 |
| --- | --- |
| [[小市值日内做T]] | 作为买入前低频过滤 |
| 小市值周频轮动 | 作为基本面质量约束 |

## 验证设计草案

先只做过滤，不进入打分，避免和质量盈利因子混杂。
