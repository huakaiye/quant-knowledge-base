# I20260523-001 多策略风险预算与组合层

## 一句话描述

把 ETF 动量、小市值、质量防御、低波等策略视作不同收益来源，在组合层分配风险预算，而不是只比较单个策略收益。

## 模块定义

| 组件 | 定义 |
| --- | --- |
| 子策略收益 | 每个策略的净值或日收益序列 |
| 风险预算 | 按目标波动率、最大回撤、相关性和状态信号分配仓位 |
| 再平衡 | 固定周期或风险偏离阈值触发 |
| 防守资产 | 现金、货币 ETF、债券 ETF 或低波策略 |

## 适用场景

- 单一动量策略在震荡市或反转市失效。
- 小市值策略在流动性收缩或尾部风险上升时需要降权。
- 多个策略收益来源不同，但同时满仓会放大总风险。

## 验证设计草案

| 项目 | 内容 |
| --- | --- |
| 基准 | 单策略独立运行 |
| 实验 | 多策略风险预算组合 |
| 指标 | 年化收益、最大回撤、收益回撤比、相关性、风险贡献 |
| 约束 | 组合层只使用上一交易日可得净值和风险指标 |

~~~
