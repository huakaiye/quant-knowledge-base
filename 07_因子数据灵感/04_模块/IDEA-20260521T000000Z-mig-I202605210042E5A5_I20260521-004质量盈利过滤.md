# I20260521-004 质量盈利过滤

## 一句话描述

在小市值或动量股票池中加入盈利质量门槛，避免策略收益来自低质量高波动样本。

## 初始代理指标

| 指标 | 用途 |
| --- | --- |
| 毛利率 | 质量代理 |
| ROE | 盈利能力代理 |
| 营收同比 | 成长和经营稳定性代理 |
| 经营现金流 | 财务质量代理，若数据可用 |

## 适用策略

| 策略 | 适用方式 |
| --- | --- |
| [[小市值日内做T]] | 买入前过滤、持仓复查 |
| 小市值周频轮动 | 多因子排序前过滤 |

## 验证设计草案

先做过滤强度消融：无过滤、宽松过滤、中等过滤、严格过滤。
