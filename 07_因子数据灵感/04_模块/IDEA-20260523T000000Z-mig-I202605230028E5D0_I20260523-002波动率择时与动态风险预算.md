# I20260523-002 波动率择时与动态风险预算

## 一句话描述

用近期可见波动率决定组合风险预算，高波动时降低暴露，低波动且趋势有效时恢复暴露。

## 模块定义

| 版本 | 定义 |
| --- | --- |
| 已实现波动版 | 使用过去 N 日收益率标准差 |
| EWMA 波动版 | 使用指数加权波动率 |
| 状态叠加版 | 波动率高且尾部风险高时额外降仓 |

## 与现有模块关系

本模块是 [[波动率缩放仓位]] 的上位框架。现有模块更偏单策略仓位缩放，本模块强调组合层风险预算和状态切换。

## 未来函数审计

- 只使用调仓日前一日及以前的数据。
- 若使用分钟数据，只能用调仓时点之前的数据。
- 不能用当日完整收盘波动率决定当日收盘前交易。

~~~
