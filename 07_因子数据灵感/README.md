# 因子数据灵感说明

本目录保存从文献、市场复盘、旧实验和平台诊断中提炼出来的可实现灵感。

## 分类

| 目录 | 内容 |
| --- | --- |
| `01_因子/` | 可计算的打分信号，例如动量、波动率、质量、资金流 |
| `02_数据/` | 新数据源、字段、代理变量或数据处理方式 |
| `03_机制/` | 市场机制解释，例如拥挤、流动性冲击、风险偏好 |
| `04_模块/` | 可嵌入策略的模块想法，例如防御门控、换仓抗抖 |

## 升级为实验的条件

灵感进入实验前必须明确：

- 关联哪个研究方向。
- 需要哪些平台字段。
- 预期改善什么指标。
- 基准对照是什么。
- 如何证伪。

没有这些条件的灵感保留为候选，不直接跑回测。
