# R010A10S1 A2 换仓延迟确认默认关闭 Shadow 观察规格

## 一句话结论

R010-A10S1 只允许作为 default-off shadow 观察层：记录“如果 A2 在低离散、小差距换仓时延迟 3 个交易日确认，会形成什么候选释放日”，但不得改变真实目标、权重、订单或成交。

本页不是实盘启用指令。

## Shadow 条件

```text
action == A2
AND final target differs from existing target
AND dispersion < 0.04
AND top1_top5_gap < 0.05
AND delay_days == 3
=> record switch delay shadow
```

当前版本不包含：

```text
delay_days = 1/2/5
old target topN guard
old-vs-new momentum guard
volatility guard
capacity guard
small-cap extension
A0/A1/all_actions extension
```

## 当前边界

| 项目 | 当前规则 |
| --- | --- |
| 默认状态 | 关闭 |
| 真实交易 | 不改写 |
| 作用域 | 仅 A2 |
| 观察对象 | A2 从旧目标切到新目标时的 3 日延迟确认 |
| 触发阈值 | dispersion < 0.04 且 top1_top5_gap < 0.05 |
| 释放日 | 从观察日向后第 3 个可用交易日 |
| 缺失字段 | 默认不允许触发 |
| 实盘授权 | 无 |

## 必看日志字段

| 字段 | 用途 |
| --- | --- |
| `observe_date` | 观察交易日 |
| `signal_date` | 信号日期 |
| `release_date` | 延迟确认释放日 |
| `delay_days` | 延迟交易日数量 |
| `live_enabled` | 必须为 false |
| `shadow_enabled` | 允许为 true |
| `action` / `applied_action_before_a10` | 必须在 A2 作用域 |
| `scope_pass` / `switch_pass` / `rule_pass` / `trigger` | 分解触发漏斗 |
| `old_target` | 原持仓目标 |
| `new_target` | 当前 A2 新目标 |
| `r010a10_dispersion` | ETF 池离散度 |
| `r010a10_top1_top5_gap` | Top1 相对 Top5 差距 |
| `risk_cap_before_a10` | 进入 shadow 前的风险仓上限 |
| `orders_changed_by_shadow` | 只代表影子目标与真实目标不同，不得真实下单 |
| `shadow_targets_if_delayed` | 影子延迟目标 |
| `actual_targets_before_a10` | 真实目标快照 |

## 交易隔离要求

| 检查 | 通过标准 |
| --- | --- |
| 启动参数 | `r010a10_a2_switch_delay_shadow_enabled=false` 为默认 |
| 真实目标 | shadow helper 不得 mutate `final_targets` |
| 权重和订单 | shadow 只写日志，不写真实 weights/orders |
| 权益曲线 | shadow-only 运行与同配置 baseline 一致 |
| 成交路径 | trades/orders 去除技术性 `order_id` 后一致 |
| release_date | 真实触发事件必须可解析到释放日 |
| 命名 | 不得生成 `R010A10_` 后缀交易标的 |

## 升级与停止规则

| 结果 | 决策动作 |
| --- | --- |
| 任一 shadow 字段改变真实 targets、weights、orders、trades | `kill_live_path` |
| 真实触发事件没有 release_date | `revise_logging` |
| 触发少于 8 个新样本 | 继续 `observe`，不得交易化 |
| 触发不足 2 个市场状态段 | 继续 `observe`，不得交易化 |
| 样本集中在单一短窗口 | `reject_live_promotion` |
| 新规则满足触发容量、四分段稳定、隔离和机制解释 | 仅允许进入下一轮 default-off shadow 候选 |

## 当前推进结论

```text
keep_default_off_shadow_no_live
```

下一步不是开启 S1，而是做 R010-A10S2 只读触发面扩展，优先从真实路径 `scope_pass AND switch_pass` 样本中寻找更大容量的 A2 延迟确认规则。
