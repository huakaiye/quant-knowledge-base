# R010B3 B3-gate默认关闭观察方案

## 一句话结论

B3-gate-median-or-breadth-cap80 已完成四段验证和 MiniQMT v18 默认关闭映射，但本页不是开启实盘指令。当前任务是把后续观察要看的日志、触发日、人工复核字段和升级边界固定下来。

## 当前边界

| 项目 | 当前规则 |
| --- | --- |
| 默认状态 | `ENABLE_B3_GATE_MOB_CAP80=0`，不改变实盘默认交易 |
| 观察前提 | 只有用户明确授权模拟盘、小资金或演练时，才允许显式打开 gate |
| 主体策略 | A2-slope004 继续作为进攻增强主线 |
| 防守候选 | B3-gate-median-or-breadth-cap80 只作为默认关闭观察候选 |
| R026 | 只作为低覆盖风险复核标记，不直接驱动仓位 |

## R026 复核边界

R026 防守池分层当前只作为解释层。最近两类审计给出的边界如下：

| 审计 | 关键结果 | 当前含义 |
| --- | --- | --- |
| B3 gate 后 R026 增量审计 | `signal_B_equity_defense` 未来 10 日 -5% 下探精度 35.09%，gate blocked raw 覆盖 0%，non raw future dd5 覆盖 5.56% | 干净但覆盖低，只适合风险复核标记 |
| R026B-light-cap90 2024 配对首段 | 只 active 3 天，相对 B3 gate 收益 -1.08pp，最大回撤仅改善 0.036pp | 自动 cap90 动作不值得继续扩大 |

后续实盘或离线复核中，`r026_marker` 只回答“这个风险日是否有外部防守池共振”，不得单独触发 cap90、cap80 或清仓动作。

## 规则口径

| 观察对象 | 判定口径 | 日志或状态字段 |
| --- | --- | --- |
| raw 全弱候选 | B3 全弱且累计全弱 `cum_days>=8` 或连续全弱 `consecutive_days>=3` | `raw_signal_active` |
| gate 通过日 | raw 全弱候选且 `median_ret<=0 OR ma20_breadth<=0.40` | `gate.pass`、`signal_active` |
| gate 拦截日 | raw 全弱候选存在，但 median/breadth 未确认 | `raw_signal_active=true` 且 `gate.pass=false` |
| active cap80 日 | gate 通过后限时 2 个交易日内实际压到 80% 风险仓 | `active`、`apply_days_left` |
| 风险仓动作 | 风险资产最高 80%，未用仓位转入 `511880.XSHG` | `B3_ALL_WEAK_PERSISTENCE_RISK_CAP=0.80`、目标权重日志 |

## 必看日志与状态字段

| 来源 | 字段或日志片段 | 用途 |
| --- | --- | --- |
| 状态文件 | `b3_all_weak_persistence_state` | 复核跨日持久计数是否连续 |
| 状态字段 | `cum_days`、`consecutive_days`、`quiet_days` | 判断信号是持续弱化还是单日噪声 |
| 状态字段 | `raw_signal_active`、`signal_active`、`trigger_today`、`active` | 区分 raw、gate、首次触发和实际 cap80 |
| 状态字段 | `apply_days_left` | 复核 limited2 是否只影响有限天数 |
| gate 字段 | `median_ret`、`ma20_breadth`、`gate.pass`、`gate.reason` | 解释 gate 是因中位收益、广度还是被拦截 |
| 实盘日志 | `[B3-cap90] 滤波快照` | 复核全弱基础信号来自 Top1、当前持仓和 Top5 防抖滤波 |
| 实盘日志 | `[B3-gate] 持久状态` | 复核 raw、active、cum、consec、quiet 和 gate 结果 |
| 实盘日志 | `[B3-gate] B3全弱持久+median/breadth确认触发` | 只在实际 cap80 目标权重生成时出现 |

## 人工复核字段

每次出现 raw、gate blocked、trigger 或 active 时，只在研究库记录摘要；完整日志和表格应放入平台结果目录。

| 字段 | 说明 |
| --- | --- |
| `observe_date` | 观察交易日 |
| `event_type` | `raw_only` / `gate_blocked` / `trigger_today` / `active_cap80` |
| `current_position` | 触发前风险持仓和防御 ETF 仓位 |
| `candidate_top1` | 当日 Top1 候选及健康状态 |
| `raw_reason` | B3 全弱原因，来自滤波快照 |
| `cum_days` / `consecutive_days` / `quiet_days` | 持久状态 |
| `median_ret` / `ma20_breadth` | gate 确认变量 |
| `gate_reason` | 是否通过 gate 及原因 |
| `target_weights` | 最终目标权重，尤其是风险仓与 `511880.XSHG` 比例 |
| `r026_marker` | 若离线复核可得，记录 `signal_B_equity_defense` 是否出现 |
| `manual_judgement` | 人工判断：有效防守、误伤趋势、漏报风险或无法判断 |

## 复核问题

| 问题 | 判断依据 |
| --- | --- |
| active cap80 日是否处在真实回撤扩张期 | 对照当日组合回撤、后续 5/10 日净值和持仓收益 |
| gate blocked raw 日是否真的只是噪声 | 观察 blocked 后 5/10 日是否未出现明显下探 |
| 是否误伤强趋势延续 | active 后原风险资产是否继续明显上涨 |
| limited2 是否按预期退出 | `apply_days_left` 是否清零，`quiet_days` 是否正常累计 |
| R026 是否只提供解释价值 | `signal_B_equity_defense` 是否集中出现在少数高风险日，而不是宽泛覆盖 |
