# R010A9S3 过滤层稀薄 Top3 默认关闭 Shadow 观察规格

## 一句话结论

R010-A9 S3 只允许进入默认关闭 shadow 观察：记录“如果把 A2 原目标替换为 Top3 会怎样”，但不得改变真实目标、真实权重、订单或 MiniQMT 默认开关。

本页不是实盘启用指令。

## Shadow 条件

```text
action == A2
AND r010b3_filter_all_weak == true
AND top2_ret5 > 0
AND median_ret <= 0.03
AND top3_code exists
AND top3_code != current A2 target
=> record shadow target Top3
```

明确不包含：

```text
S4 hot_branch_v1
top2_ret5 >= 0.005 收窄
median_ret <= 0.020 收窄
no_repeat_within_N_days
Top3 score guard
target-vs-Top3 ret5 guard
```

## 当前边界

| 项目 | 当前规则 |
| --- | --- |
| 默认状态 | 关闭 |
| 真实交易 | 不改变 |
| 作用域 | 仅 A2 |
| 观察对象 | A2 原目标 vs Top3 影子目标 |
| 资金权重 | 影子权重沿用当前 A2 风险仓上限 |
| 禁止项 | 不扩展小市值；不扩展 A0/A1；不替代防守模块；不后验换参 |

## 必看日志字段

| 字段 | 用途 |
| --- | --- |
| `observe_date` | 观察交易日 |
| `signal_date` | 信号日期 |
| `live_enabled` | 必须为 false |
| `shadow_enabled` | 允许为 true |
| `action` / `applied_action_before_a9` | 必须在 A2 作用域 |
| `rule_pass` / `scope_pass` / `trigger` | 判断是否产生 shadow 事件 |
| `target_before_a9` | A2 原风险目标 |
| `top3_code` | 影子替代目标 |
| `actual_targets_before_a9` | 真实目标 |
| `shadow_targets_if_top3` | 影子目标 |
| `orders_changed_by_shadow` | 只表示影子与真实目标不同，不得真实下单 |
| `risk_cap_before_a9` | 影子使用的风险仓上限 |
| `median_ret` / `top2_ret5` / `dispersion` | 规则解释字段 |
| `r010a9_target_*` | 原目标 rank、ret20、ret5、slope10、r2、score |
| `r010a9_top2_*` | Top2 动量上下文 |
| `r010a9_top3_*` | Top3 动量上下文 |
| `r010a9_top2_minus_top3_*` | Top2 与 Top3 间距 |
| `r010a9_target_minus_top3_*` | 原目标与 Top3 间距 |

## 交易隔离要求

| 检查 | 通过标准 |
| --- | --- |
| 启动日志 | `r010a9_filter_thin_top3_replace_enabled=false` |
| 真实目标 | `actual_targets_before_a9` 不被 shadow 改写 |
| 订单 | trades/orders 的经济内容与同配置 default-off 基线一致；运行生成的 `order_id` 可不同 |
| 权益曲线 | shadow-only 运行与 default-off 基线一致 |
| 日志 | 触发事件有完整 `r010a9_*` 上下文字段 |
| 异常 | 缺字段、缺目标、目标停牌、订单差异均记为失败 |

## 事件复核

每个 shadow 触发需要人工复核：

| 项目 | 判断 |
| --- | --- |
| 原目标是否可成交 | 是否停牌、涨跌停、无成交 |
| Top3 是否可成交 | 是否停牌、涨跌停、无成交 |
| 替代是否有成交成本优势 | 换手、滑点、最小佣金影响 |
| 后续 1/3/5/10/20 日表现 | 影子目标相对原目标和组合基线 |
| 是否发生路径依赖 | 连续触发是否改变下一次起点 |
| 是否属于已否决模式 | S4 热分支或简单收窄误删模式 |

## 升级与停止规则

| 结果 | 决策动作 |
| --- | --- |
| 任一 shadow 字段改变真实目标、真实权重或订单 | `kill_live_path` |
| 事件字段无法还原原目标、Top3 或风险权重 | `revise_logging` |
| 触发少于 8 个新样本 | 继续 `observe` |
| 新样本不足 2 个市场状态段 | 继续 `observe` |
| 扣除滑点、最小佣金和可成交性后贡献为负 | `reject_live_promotion` |
| 连续触发路径依赖吞噬贡献 | `reject_or_split_cluster_rule` |
| 满足样本数、交易隔离、成交可用性和净贡献要求 | 仅允许进入下一轮预注册，不得直接开启实盘 |

## 当前推进结论

```text
promote_to_shadow_observation_no_live
```

下一步不是新增 S5 规则，而是让 R010-A9 S3 进入默认关闭 shadow 观察规格。观察通过前，不讨论实盘开关。
