# 多策略市场状态切换方法库

本方法库定义未来多策略切换研究的默认流程。核心原则是：先证明市场状态能解释策略差异，再讨论策略切换；先做日志和归因，再接入交易。

## 研究顺序

| 阶段 | 名称 | 是否交易 | 目的 |
| --- | --- | --- | --- |
| S0 | 文献和机制整理 | 否 | 明确每类策略赚什么风险 |
| S1 | 状态日志 | 否 | 逐日记录市场环境标签 |
| S2 | 策略适配归因 | 否 | 检查不同状态下策略表现是否显著不同 |
| S3 | 风险预算降权 | 是，但只降权 | 在不切换策略的情况下降低失效期风险 |
| S4 | 策略路由 | 是 | 根据状态分配不同策略权重 |
| S5 | 样本外和仿真 | 是 | 判断是否进入实盘观察 |

## 策略收益来源表

| 策略类别 | 主要收益来源 | 有利环境 | 不利环境 |
| --- | --- | --- | --- |
| 小市值 | 流动性补偿、风险偏好扩散、低覆盖错定价 | 广度高、成交扩张、小盘相对强 | 流动性压力、退市风险、窄主线 |
| ETF 动量 | 趋势延续、横截面强弱分化 | 趋势强、离散度高、TopN 有动力 | 低离散震荡、急跌后反弹 |
| 质量/价值 | 盈利质量、估值修复、风险重估 | 盈利分化、质量重估、风险偏好中低 | 纯题材、强风险偏好末端 |
| 低波/防守 | 避险需求、低 beta、风险预算下降 | 高波动、流动性压力、弱势无主线 | 全面风险偏好上行 |
| Carry/红利 | 持有收益、现金流、稳定性 | 利率下行、风险偏好中低、震荡市场 | 强成长主线、利率上行冲击 |

## 状态识别信号

| 信号族 | 指标 | 主要区分 |
| --- | --- | --- |
| 趋势 | 宽基收益、均线斜率、指数位置 | 进攻还是防守 |
| 广度 | 上涨占比、MA20 上方占比、行业扩散 | 普涨还是窄主线 |
| 风格 | 小盘/大盘、成长/价值、红利/成长 | 哪类策略优先 |
| 离散 | 截面标准差、Top1-Top5 分差 | 轮动是否值得做 |
| 流动性 | 成交额、跌停占比、左尾收益、小票成交 | 小市值是否可交易 |
| 策略健康 | 子策略回撤、波动、换手、连续失效天数 | 是否降权 |

## 决策规则设计

| 规则 | 要求 |
| --- | --- |
| 状态进入 | 常规状态至少连续 2 日确认，强风险状态可单日触发 |
| 状态退出 | 风险状态退出需要趋势和广度同时修复 |
| 切换冷却 | 策略权重切换后设置最短观察期 |
| 权重变化 | 第一阶段只允许阶梯式降权，不允许频繁连续调权 |
| 风控优先 | 流动性压力和尾部风险信号优先级高于收益信号 |

## 实验前预测模板

正式实验前必须写清楚：

- 哪个状态应当使哪个策略受益。
- 哪个状态应当使哪个策略失效。
- 状态路由主要改善收益、回撤、换手还是尾部风险。
- 哪些观察会证明状态路由只是事后拟合。

## 证伪测试

| 测试 | 目的 |
| --- | --- |
| 状态内收益差异 | 判断状态是否真的区分策略适配度 |
| 去极端日 | 判断结果是否只来自少数暴涨暴跌日 |
| 成本后表现 | 判断切换是否被换手吞噬 |
| 延迟一日执行 | 判断是否依赖不可交易的同步信号 |
| 消融测试 | 判断趋势、广度、风格、流动性哪个信号有贡献 |
| 样本外年份 | 判断阈值是否过拟合 |

## 不允许的做法

- 不允许用未来收益定义市场状态。
- 不允许用全样本分位数设置状态阈值。
- 不允许看到某策略最近亏损就直接关闭。
- 不允许一次性引入太多状态和参数。
- 不允许把策略路由收益归因给状态，而不检查 beta、行业、市值和流动性暴露。
