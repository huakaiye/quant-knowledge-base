# 实验记录

本目录保存每一次研究记录。研究记录不是每一次回测，而是每一次有明确假设的研究尝试。

## 命名规则

```text
RYYYYMMDD-序号_简短主题.md
```

示例：

```text
R20260521-001_ETF双池波动率过滤.md
```

## 新建流程

1. 复制 [[研究记录模板]]。
2. 修改 `research_id`、标题、双标签和目标策略。
3. 保存实验配置到 `configs/research/<research_id>/`。
4. 运行回测并保存输出到 `results/v2/research/<research_id>/`。
5. 在研究记录中填写关键对比表和结论。
6. 打开 [[03-研究台账]] 检查是否自动汇总。

## 状态定义

| 状态 | 含义 |
| --- | --- |
| `草稿` | 只有想法或初步配置，尚未完成基准对照 |
| `观察中` | 已有结果，但仍缺滚动验证、样本外验证或消融 |
| `已采纳` | 通过对照、消融和样本外验证，可进入策略路线 |
| `已废弃` | 结果不稳定、逻辑无效或风险收益不达标 |

~~~
