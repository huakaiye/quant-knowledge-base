# 涨停事件价量结构专题

本专题用于研究[[涨停事件]]之后的短线收益是否可以被涨停前[[价量结构]]解释，以及这类策略在加入[[集合竞价成交模型]]、涨跌停不可成交、容量和样本外约束后是否仍有研究价值。

这个方向暂不进入交易主线。它的第一目标是把聚宽社区中的“价量因子核心窗口”从主观形态和极端回测收益，转化为可证伪、可审计、可样本外验证的研究问题。

## 核心问题

| 层级 | 问题 | 输出 |
| --- | --- | --- |
| 事件基准 | 非一字涨停股次日买入、持有 2 日是否有正期望 | 涨停事件基准收益 |
| 价量增量 | 涨停前价量结构是否能提高事件后收益回撤比 | 价量结构因子 |
| 执行真实度 | 次日集合竞价是否可成交，容量能支持多少资金 | 成交容量模型 |
| 稳健性 | 样本外、年份切分、去极端样本后是否仍有效 | 研究决策 |

## 初始研究假设

涨停不是买入理由本身。值得研究的是：涨停前的价量结构是否已经显示资金准备、筹码交换和波动收敛；涨停日只是突破确认。如果该机制成立，价量结构过滤后的涨停事件应比普通涨停事件有更好的短期收益、回撤和尾部风险。

## 研究路线

| 阶段 | 名称 | 是否交易 | 目的 |
| --- | --- | --- | --- |
| S0 | 文献和策略文章归档 | 否 | 把社区文章拆成可验证模块 |
| S1 | 涨停事件基准 | 否 | 检验全部涨停事件本身是否有正期望 |
| S2 | 价量结构单因子 | 否 | 分别验证缩量整理、放量突破、波动收敛等结构 |
| S3 | 多结构组合 | 否 | 检验组合后是否只是过拟合 |
| S4 | 成交容量模型 | 否 | 加入集合竞价、涨跌停和参与率约束 |
| S5 | 小资金可交易模拟 | 是，仍需观察 | 只在通过样本外和容量审计后考虑 |

## 初始因子候选

| 因子族 | 说明 | 未来函数风险 |
| --- | --- | --- |
| 涨停前强度 | 涨停前 N 日累计收益、冲高能力、阶段位置 | 只能用涨停日收盘前已发生数据 |
| 波动收敛 | 涨停前振幅、真实波幅、上影线是否收敛 | 不能用涨停后数据定义窗口 |
| 量能结构 | 放量试盘、缩量整理、涨停日放量突破 | 成交量必须按当日可见时点处理 |
| 收盘位置 | 收盘价相对当日高低区间位置 | 若次日买入，可用涨停日收盘数据 |
| 流动性容量 | 涨停日成交额、次日集合竞价成交额代理 | 必须限制参与率 |

## 证伪重点

- 如果全部涨停事件本身没有正期望，价量结构必须证明有显著增量。
- 如果价量结构只在作者样本窗口有效，则不升级。
- 如果加入一字板、涨跌停不可成交和集合竞价容量后收益消失，则不进入策略开发。
- 如果收益集中在少数极端标的或年份，则只能保留为观察性研究。
- 如果多结构组合显著优于单结构但样本外失效，优先判定为过拟合。
