# R010B5R4 非线性回撤预算默认关闭影子观察方案

## 一句话结论

R010-B5R4 只观察非线性回撤预算建议的风险仓上限，不改变当前实盘策略的目标权重、订单或默认开关。

本页不是实盘启用指令。

## 当前边界

| 项目 | 当前规则 |
| --- | --- |
| 默认状态 | 非线性回撤预算不得进入实盘默认交易逻辑 |
| 主体策略 | 当前实盘主体仍是 A2-slope004 + B3 gate + B3-gate-tiered-v2 |
| 观察对象 | `nonlinear_suggested_cap` 与 `actual_tiered_cap` 的差异 |
| 观察前提 | B3 gate active 且 tiered-v2 已给出实际 cap |
| 实际仓位边界 | 真实 `target_weights_actual` 必须完全不受非线性建议影响 |
| 禁止项 | 不得替代 B3 gate；不得替代 tiered-v2 风险识别；不得后验换参 |

## 与 R010-B4 影子观察的区别

| 项目 | R010-B4 tiered-v2 影子观察 | R010-B5R4 非线性影子观察 |
| --- | --- | --- |
| 观察问题 | tiered-v2 候选 cap70/cap60 是否合理 | 非线性建议 cap 是否优于实际 tiered cap |
| 基准 cap | B3 gate cap80 | 实际 tiered cap |
| 影子 cap | tiered_candidate_cap | nonlinear_suggested_cap |
| 核心差异 | `candidate_cap - 80%` | `nonlinear_suggested_cap - actual_tiered_cap` |
| 最大风险 | 影子 cap 变成真实 cap | suggested cap 改写真实 target weights |

## 必看日志与状态字段

| 来源 | 字段或日志片段 | 用途 |
| --- | --- | --- |
| 启动日志 | 非线性预算交易开关 | 必须确认未真实启用 |
| R010-B ACTION | `r010b3_gate_tiered_strength_level` | 获取 tiered 等级 |
| R010-B ACTION | `r010b3_gate_tiered_strength_effective_cap` / `r010b3_strength_effective_cap` | 获取实际 tiered cap |
| R010-B5 状态日志 | 组合权益、峰值、回撤 | 解释非线性建议 |
| R010-B5 预算日志 | threshold、decay、floor、risk_scale | 解释建议 cap |
| 影子日志 | `nonlinear_suggested_cap`、`cap_delta` | 核心观察字段 |
| 交易日志 | target weights、orders | 证明影子未改交易 |

## 人工复核字段

| 字段 | 说明 |
| --- | --- |
| `observe_date` | 观察交易日 |
| `event_type` | `shadow_nonlinear_lower_than_tiered` / `shadow_nonlinear_same_as_tiered` / `shadow_nonlinear_higher_than_tiered` |
| `b3_gate_active` | 必须为 true 才有观察意义 |
| `tiered_level` | `normal`、`strong`、`extreme` |
| `actual_tiered_cap` | 当前实盘或研究基准实际 cap |
| `nonlinear_suggested_cap` | 非线性预算建议 cap |
| `cap_delta` | 非线性建议和实际 cap 的差 |
| `portfolio_drawdown` | 组合回撤 |
| `risk_scale` | 非线性缩放倍率 |
| `target_weights_actual` | 真实目标权重 |
| `target_weights_if_nonlinear` | 影子目标权重 |
| `orders_changed_by_shadow` | 观察期必须为 false |
| `next_1d` / `next_5d` / `next_10d` | 后验表现 |
| `manual_judgement` | 有效保护、误伤、削弱防守、无法判断 |

## 复核问题

| 问题 | 判断依据 |
| --- | --- |
| 非线性建议是否只在 B3/tiered 风险窗口内出现 | `b3_gate_active` 与 `tiered_level` |
| 建议更低 cap 是否保护回撤 | 后续 1/5/10 日组合与风险资产收益 |
| 建议更低 cap 是否误伤反弹 | 风险资产后续快速修复 |
| 建议更高 cap 是否削弱强风险防守 | strong/extreme 后续回撤是否加深 |
| shadow 是否完全不改仓位 | `target_weights_actual`、orders、`orders_changed_by_shadow=false` |
| 建议 cap 是否可解释 | 回撤、threshold、decay、floor、risk_scale 是否齐全 |

## 升级与停止规则

| 结果 | 决策动作 |
| --- | --- |
| 任一 shadow 字段改变真实 target weights 或订单 | `kill_live_path` |
| 日志无法还原建议 cap 或实际 cap | `revise`，先补字段 |
| 非线性绕过 B3 gate/tiered-v2 独立触发 | `kill_partial_replacement` |
| 差异事件少于 8 天 | 继续 `observe` |
| 差异事件胜率低于 75% 或累计 log edge 为负 | `revise` |
| 误伤快速修复不可解释 | `revise` 或 `kill` |
| 满足样本、胜率、edge 和交易隔离要求 | 仅允许进入 R010-B5R5 预注册 |
