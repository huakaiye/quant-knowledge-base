# 模块候选看板

自动聚合 `💡 想法/` 目录下所有 `type: 想法` 的模块候选。想法状态变化时自动更新，无需手动维护本页。

```base
filters:
  and:
    - file.inFolder("💡 想法")
    - type == "想法"
properties:
  file.name:
    displayName: 文件
  idea_id:
    displayName: 想法编号
  status:
    displayName: 状态
  priority:
    displayName: 优先级
  module_type:
    displayName: 模块类型
  source_tags:
    displayName: 收益来源
  execution_tags:
    displayName: 执行方式
  related_strategies:
    displayName: 适用策略
  data_requirements:
    displayName: 数据需求
  difficulty:
    displayName: 实现难度
  expected_effect:
    displayName: 预期作用
  related_literature:
    displayName: 来源文献
  next_action:
    displayName: 下一步
views:
  - type: table
    name: 全部想法
    order:
      - idea_id
      - status
      - priority
      - module_type
      - related_strategies
      - expected_effect
      - next_action
    columnSize:
      file.name: 220
  - type: table
    name: 高优先级
    filters:
      and:
        - priority == "高"
    order:
      - idea_id
      - module_type
      - related_strategies
      - expected_effect
      - next_action
  - type: table
    name: 最高优先级
    filters:
      and:
        - priority == "最高"
    order:
      - idea_id
      - module_type
      - related_strategies
      - expected_effect
      - next_action
```

## 状态口径

| 状态 | 含义 |
| --- | --- |
| `待研究` | 已拆成模块，等待确定实验设计 |
| `研究中` | 已进入预实验或文献验证阶段 |
| `待实验` | 设计完成，等待平台可用后回测 |
| `已转研究` | 已建立正式研究记录 |
| `已废弃` | 不适合当前平台或证据不足 |
