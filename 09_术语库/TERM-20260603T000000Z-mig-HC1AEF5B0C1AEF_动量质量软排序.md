# 动量质量软排序

## 定义

动量质量软排序（Momentum Quality Soft Ranking）是指：**不直接剔除动量质量不足的候选，而是将质量维度作为排序时的加分项、扣分项或权重调整因子**。所有候选仍然参与排序，但质量好的得分更高，质量差的得分更低。

## 与硬准入的核心对比

| 维度 | 软排序 | 硬准入 |
|------|-------|--------|
| 失败候选的命运 | 降分，仍留在池中 | 直接剔除 |
| 候选池规模 | 不变 | 可能缩减 |
| 对极端行情的容忍度 | 高 | 低（可能空仓） |
| 误删风险 | 无 | 有（可能在阈值边界误删好候选） |
| 信号利用效率 | 高（所有候选信息被利用） | 低（淘汰候选的信息被丢弃） |

## 三种加权方式

### 1. 乘法衰减
```text
final_score = raw_momentum_score × quality_multiplier
quality_multiplier ∈ [0.7, 1.3]
```

### 2. 分位重排
```text
final_rank = w_m × rank(z_score(momentum)) + w_q × rank(z_score(quality))
```

### 3. 门槛加分
```text
bonus = (quality > threshold) ? fixed_bonus : 0
final_score = raw_momentum_score + bonus
```

## 避免过度剔除的好处

1. **保留候选池多样性**：ETF 双池仅 10-15 个品种，软排序确保 Top N 总能选出来
2. **避免阈值悬崖效应**：硬准入在阈值附近存在"差一分就淘汰"的悬崖
3. **信息利用效率更高**：保留了全部候选的动量信息，只调整优先级
4. **适应性更强**：连续函数比二元判断更易泛化到不同市场状态

## 在实际策略中的体现

[[动量R2得分]] 本身就是软排序思维的体现——`年化回归收益 × R²` 同时衡量动量强度和质量，无需先做"R² > 0.3 才准入"的硬过滤。[[候选排序惩罚]] 是软排序的另一种形态。
