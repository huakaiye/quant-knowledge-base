# 状态日志

状态日志是指策略在不改变交易动作的前提下，逐日记录市场环境、候选池结构、风险信号和建议模式的研究数据。

在当前研究库中，它主要服务 [[07-市场状态识别专题]] 和 [[ETF双池动量轮动]]：先用日志验证状态信号是否稳定、是否领先主要回撤、是否存在未来函数，再决定是否进入防守门控或策略切换实验。

## 常见字段

| 字段 | 用途 |
| --- | --- |
| `signal_date` | 标记信号可用日期，审计未来函数 |
| `raw_state` | 当日原始状态 |
| `confirmed_state` | 经确认器后的状态 |
| `ret20_breadth` | ETF 池 20 日正收益广度 |
| `ma20_breadth` | ETF 池 `close > MA20` 广度 |
| `top1_ret5` | 最强候选近 5 日动量 |

## 使用原则

- 第一阶段只记录，不交易。
- 不用全样本分位数生成当日状态。
- 任何交易门控必须先经过状态日志的样本外审计。

~~~
