# R20260531-022 R010C动作启动信号诊断与重构

## 研究背景

R010-B1、R010-B2、R010-B3 已分别验证 A1、A2、A3/A4 的轻降仓、抗抖和持久防御动作。阶段性结论很明确：这些动作尚未对 ETF 双池轮动形成稳定有效改进。

当前更应该质疑的不是动作力度，而是启动方式：

| 动作 | 原意 | 当前问题 |
| --- | --- | --- |
| A1 | 弱势时轻降仓或递补候选 | 能降低一点回撤，但复合收益代价明显 |
| A2-current | 当前旧仓仍有动量则抗抖 | 2024有效，但2020-2021和2025-2026明显拖累 |
| A2-env046 | 高广度强主线时抗抖 | 复合收益提高，但收益集中在2024，非稳定防守 |
| A3/A4单日 | 强弱势信号触发防守 | 触发极少，无法覆盖连续回撤 |
| A3/A4持久防御 | 触发后维持多日半防御 | 2024回撤只改善约1pp，收益下降，不达长周期门槛 |

因此 R010-C 不再问“动作如何调参”，而是问：

> 这些启动信号是否真的能提前识别未来需要降仓、抗抖、防守或保持轮动的市场状态？

## 核心假设

如果某个动作启动信号有价值，它在触发后的未来窗口内应当表现出可观测的预测能力：

| 信号类型 | 它应该预测什么 | 失败表现 |
| --- | --- | --- |
| 降仓信号 | 未来组合回撤或负偏度上升 | 触发后继续上涨，形成误伤 |
| 抗抖信号 | 旧仓相对新候选继续占优 | 触发后旧仓跑输新候选 |
| 防守信号 | 风险资产未来回撤显著高于防御资产 | 触发后防御资产拖累收益 |
| 恢复信号 | 未来趋势恢复且风险资产胜率上升 | 过早退出防御或过晚恢复 |

## 实验前预测

| 预测 | 可观察指标 | 决策含义 |
| --- | --- | --- |
| 有效启动信号应具备正向标签命中率 | 触发后5/10/20日风险事件发生率高于非触发日 | 可继续设计交易动作 |
| A2类信号必须解释旧仓相对候选优势 | 旧仓未来收益减新Top1/Top3候选收益为正 | 才允许抗抖保留 |
| A3/A4类信号必须解释防御资产相对优势 | 风险资产未来回撤大于防御资产机会成本 | 才允许持久防御 |
| 若信号只在2024有效 | 分段表现集中在单一年份 | 标记为特定市场假设，不进入通用组合 |

| future_mdd_5/10/20 | 信号日后组合或目标ETF窗口最大回撤 | 判断降仓/防守是否必要 |

| opportunity_cost | 动作版本收益 - B0收益 | 量化误伤成本 |

- A2 不是稳定的“旧仓优于新 Top1”信号。A2 多数时候只是继续持有 Rank1，因此“抗抖”并没有真正被检验；当它确实选择非 Rank1 时，旧仓 20 日后平均比 Top1 低 2.22pp。
- A2 失败候选中，Top1、Top3 与旧仓一起走弱，问题不是“换错仓”，而是整个风险资产池进入了高波动弱化窗口；此时 511880 相对旧仓平均有 +1.44pp 超额。
- A0 漏防候选中，旧仓、Top1、Top3 都转负，511880 相对旧仓平均有 +2.46pp 超额。这说明当前最值得研究的不是“保留谁”，而是“什么时候 Top1-Top5 已经不该持有风险资产”。
- A3/A4 样本仍各只有 1 天，不能继续作为主要证据。若后续要防守，应从 A0 漏防和 A2 失败样本中反推可见门控，而不是沿用原 A3/A4 单日触发方式。
