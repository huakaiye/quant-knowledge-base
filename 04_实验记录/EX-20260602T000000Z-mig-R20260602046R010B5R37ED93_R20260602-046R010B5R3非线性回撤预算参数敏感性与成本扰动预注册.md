# R20260602-046 R010B5R3 非线性回撤预算参数敏感性与成本扰动预注册

## 研究背景

R010-B5R2 叠加版 `nl_endo_gate_any_d10_a5_f70` 已完成四段真实路径验证：

| 指标 | 结果 |
| --- | ---: |
| 四段拼接复合收益 | +1534.91% |
| 四段拼接最大回撤 | -25.33% |
| 相对 B3-gate-tiered-v2 收益 | +66.57pp |
| 相对 B3-gate-tiered-v2 回撤改善 | +1.80pp |
| 动作日胜率 | 77.78% |

但这个结果仍可能来自单点参数偶然，尤其是：

```text
drawdown_threshold = 10%
decay = 5
floor = 70%
```

因此 R010-B5R3 的目标不是继续找更优参数，而是验证当前候选附近是否稳定。

## 核心假设

主假设：

```text
如果非线性组合回撤预算确实有效，那么在 B3 gate/tiered-v2 已确认风险窗口内，
阈值、衰减和最低缩放的相邻扰动不应导致收益或回撤大幅崩坏。
```

成本假设：

```text
如果该模块的优势不是交易成本幻觉，那么在更高佣金和轻度滑点压力下，
相对 B3-gate-tiered-v2 的收益保留与回撤改善仍应大体成立。
```

替代边界假设：

```text
本轮不直接测试替代 B3 gate 或替代 tiered-v2。
R010-B5R2 的 replacement 版本已经反证：组合回撤预算不能替代风险强度识别。
若 R010-B5R3 通过，只能支持它作为二级预算候选继续推进。
```

## 2026-06-02补充：部分替代边界

用户目标允许在效果足够好时考虑顶替原有的部分防御模块。该目标需要严格限定为“部分替代”，不能扩大解释：

```text
不得替代 B3 gate 的风险窗口识别。
不得替代 tiered-v2 的强/极端风险识别条件。
不得直接替代实盘防御模块。
```

若 R010-B5R3 四段参数敏感性与成本扰动通过，下一步只能考虑两条默认关闭研究路径：

| 路径 | 含义 | 前置条件 |
| --- | --- | --- |
| R010-B5R4 影子观察边界 | 保留现有实盘交易逻辑，只记录非线性预算建议 cap 与实际 tiered cap 的差异 | R010-B5R3 四段通过 |
| R010-B5R5 部分替代预注册 | 仅在 B3 gate active 且 tiered 已识别 strong/extreme 后，测试用非线性预算替代固定 cap70/cap60 的“仓位强度表达”，不替代触发识别 | R010-B5R4 影子观察显示建议 cap 稳定且误伤可控 |

这里的“顶替”不是上线承诺，而是后续可证伪实验问题。若四段敏感性或成本扰动失败，则该部分替代路径直接停止。

## 竞争性解释

需要主动证伪的解释：

| 竞争解释 | 验证方式 |
| --- | --- |

## 四段结果解释预设

四段结果出来后，解释顺序固定如下，避免看到结果后改口径：

1. 先看数据完整性：40 个配置是否均有唯一 summary，是否有异常返回码、重复 job、缺失 equity/trades/orders。
2. 再看成本扰动：`comm2x` 与 `comm2x_slip2bps` 必须在同成本基准下比较，不能拿原始成本基准解释高成本候选。
3. 再看参数邻域：6 个相邻扰动是稳健性证据，不是选新最优参数的扫描。即使某个扰动优于主候选，也不能后验切换主参数。
4. 再看分段稳定性：重点检查 2025-20260519 是否出现超过 -5pp 的收益拖累，不能只用 2024 主回撤窗口解释全部优势。
5. 最后才讨论路线：若四段通过，也只是允许进入 R010-B5R4 默认关闭影子观察；若失败，则停止部分替代路径或降为观察。
