# R20260530-016 R010-A 市场状态日志首轮实现

## 研究信念

关联信念：[[BLF20260523-001_市场状态识别与动量门控研究信念]]

当前相信的机制：ETF双池动量轮动的收益质量依赖市场状态。**但R014和R015已证明外部基准（510300、分桶基准）不能有效捕捉ETF双池的风险状态。** R010-A用策略池内生指标（趋势广度、Top5动力、横截面离散度）来判断市场状态，从源头上解决了风险代理失配问题。

先验置信度：中

## 前置实验教训

| 前置 | 教训 | R010-A的应对 |
|------|------|-------------|
| R014 分桶过滤 | 外部基准×9仍无法捕捉策略真正风险 | 全部使用策略池内生指标 |
| R015 510300二元开关 | 510300与ETF双池持仓β相关性太低 | 不看510300，看策略池自身 |
| 两个方向都被kill | 防御需要正确的风险代理 | 先验证状态标签本身是否有解释力 |

## 本次研究假设

用策略池内生指标（趋势广度、中位20日收益、Top5动力、横截面离散度）构建的7种市场状态，能够区分ETF双池策略在不同市场环境下的收益质量。**本阶段不改变任何交易行为，只验证状态标签的稳定性和解释力。**

## 改动内容

在ETF双池原策略中插入13:04的 `record_market_state` 回调：
- 从策略池内所有ETF获取历史日线数据
- 计算9个核心信号
- 按优先级判定7种原始状态
- 经确认逻辑后输出日志

**完全不修改交易逻辑。**

## 核心信号

| 信号 | 来源 | 目的 |
|------|------|------|
| 趋势广度 | ETF池中 close>MA20>MA60 比例 | 全面上涨 or 弱势 |
| 中位20日收益 | ETF池20日收益中位数 | 策略池整体方向 |
| 宽基20日收益 | 510300 20日收益 | 大盘背景（仅参考） |
| Top1绝对动量 | 池内20日收益排名第一的收益 | Top1是否真强 |
| Top5动力数量 | Top5中有绝对动量(>0)的数量 | 候选池整体动力 |
| Top1-Top5分差 | Top1收益 - Top5收益 | 排名优势是否显著 |
| 横截面离散度 | 池内20日收益标准差 | 轮动是否有选标价值 |
| 实现波动率 | ETF池等权20日波动率 | 风险水平 |
| 急跌反弹评分 | 宽基60日跌15%+近5日反弹3%+高波动 | 动量崩盘风险 |

## 状态定义

| 状态 | 条件 | suggested_mode |
|------|------|:---:|
| `crash_rebound` | 急跌+急反弹+高波动 | reduced_exposure |
| `weak_no_leader` | 中位收益< -2% + Top5动力=0 | defensive |
| `broad_uptrend` | 广度>45% + 中位收益>2% + 宽基>0 | raw_momentum |
| `narrow_leader` | 离散度>10% + Top5动力≥1 | raw_momentum_light_quality |
| `rotational_momentum` | 离散度5-25% + Top5动力≥2 + 分差>2% | smooth_soft_rank |
| `low_dispersion_chop` | 离散度<8% + 分差<3% + 中位收益≈0 | low_turnover_hold |
| `neutral_observe` | 以上都不满足 | observe |
