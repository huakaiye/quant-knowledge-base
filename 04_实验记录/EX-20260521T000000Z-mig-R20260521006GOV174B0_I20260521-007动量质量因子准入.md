# I20260521-007 动量质量因子准入

## 一句话描述

在相对动量排序之后，用“上涨是否平滑、是否多周期一致、是否接近阶段强势区、市场环境是否支持”判断 Top1-Top5 是否仍有可交易的上涨动力。

## 模块定义

当前实现为 Top5 扫描：

- 先按原策略 `年化回归收益 * R²` 排序。
- 从 Top1 到 Top5 逐个计算动量质量分。
- 第一个质量分合格的 ETF 入选。
- Top5 全部不合格时转入防御 ETF。

可拆因子：

| 因子 | 说明 | R006 结论 |
| --- | --- | --- |
| 中期动量跳过近端 | 跳过最近几日的中期收益 | 暂不采纳 |
| 平滑上涨 | 上涨天数占比高、跳涨集中度低 | 优先推进 |
| 均线排列 | MA5、MA10、MA20 多头排列加权 | 新增待测 |
| 阶段高点接近度 | 当前价接近阶段高点 | 暂不采纳 |
| 多周期趋势一致 | 20/60/120 日收益方向一致 | 暂不采纳 |
| ETF池市场状态 | 池内趋势广度和Top20动量 | 暂不采纳 |

## 下一步

- 设计软排序版本：`adjusted_score = raw_score * (0.85 + 0.30 * smooth_quality)`。

- 只有当前五候选原始动量和平滑质量都弱时，才进入防守。
- 验证 2025、2026YTD、2022-2024 是否同时不弱于基准。
