# R20260603-048 R010B5R5 非线性回撤预算严格Counterfactual预注册

## 研究信念

关联信念：[[BLF20260523-003_波动率管理与条件降仓研究信念]]

当前相信的机制：

```text
非线性组合回撤预算不应替代风险窗口识别。
它只可能在 B3 gate / tiered-v2 已经确认风险窗口后，
替代固定 cap70/cap60 的“仓位强度表达”。
```

先验置信度：中。

## 研究背景

R010-B5R4 四段 shadow-only 审计已完成，结论为：

```text
observe_counterfactual_prereg_no_live
```

已证明：

但 R010-B5R4 的事件代理只来自真实 B3-gate-tiered-v2 净值曲线，不能证明采用 `target_weights_if_nonlinear` 后的真实收益差。

因此本轮必须进入严格 counterfactual，而不是直接宣布部分替代。

## 本次研究假设

主假设：

```text
在非零 cap_delta 事件日，
若把真实目标权重 target_weights_actual 替换为影子目标权重 target_weights_if_nonlinear，
后续 1/5/10 个交易日的局部 close-to-close 组合收益差大于 0，
则 nonlinear suggested cap 对“仓位强度表达”提供了独立增益。
```

部分替代假设：

```text
如果 counterfactual edge 稳定为正，后续才允许进入 R010-B5R6：
只测试在 B3/tiered 已触发时，用 nonlinear suggested cap 替代固定 cap70/cap60。
不得替代 B3 gate。
不得替代 tiered-v2 strong/extreme 风险识别。
```

## 反事实公式

事件字段：

```text
target_weights_actual
target_weights_if_nonlinear
observe_date
```

价格字段：

```text
quant.jq_bar_daily.symbol
quant.jq_bar_daily.datetime
quant.jq_bar_daily.close_price
```

代码映射：

```text
159892.XSHE -> symbol='159892', exchange='SZ'
511880.XSHG -> symbol='511880', exchange='SH'
XBJE -> exchange='BJ'
```

对每个事件和 horizon：

```text
asset_ret_i(h) = close_i(T+h) / close_i(T) - 1
actual_ret(h) = sum(actual_weight_i * asset_ret_i(h))
shadow_ret(h) = sum(shadow_weight_i * asset_ret_i(h))
counterfactual_edge(h) = shadow_ret(h) - actual_ret(h)
```

其中 `T+h` 是观察日后第 h 个可用交易日，不做日期近似。

## 可观察预测

如果假设成立，预期应观察到：

- 预测 1：27/27 非零 `cap_delta` 事件能完整计算 1/5/10 日局部反事实收益。
- 预测 2：10 日 `counterfactual_edge` 正率不低于 70%。
- 预测 3：10 日累计 log edge 大于 0。
- 预测 4：2022-2023 与 2024 两个主要事件分段均为正 edge。
- 预测 5：2020-2021 与 2025-20260519 因事件数过少，不作为单独 promote 依据，但不能出现明显大额反向 edge。

## 竞争性解释

| 竞争性解释 | 为什么可能成立 | 如何区分 |
| --- | --- | --- |
| 事件代理只是后验看跌 | 真实净值后续亏损不等于影子权重会更好 | 直接比较 actual/shadow 权重组合收益 |
| 货币 ETF 贡献过小 | lower-cap 多数只是风险 ETF 权重小幅转到 511880 | 统计 edge 均值、分位数和 cap_delta 大小关系 |
| 交易成本吞噬 edge | 本轮 close-to-close 不含成本 | 本轮只用于晋级预注册；后续 R010-B5R6 必须加成本 |
| 分段过拟合 | 事件集中在 2022-2024 | 分段统计，2020/2025 不作为强证据 |
| 价格口径不等于实际成交 | 回测在 13:10 调仓，close-to-close 是局部近似 | 明确只作为局部 counterfactual，不等价完整回测 |
