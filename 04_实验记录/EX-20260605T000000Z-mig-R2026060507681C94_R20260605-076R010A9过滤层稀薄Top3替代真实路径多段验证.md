# R20260605-076 R010A9 过滤层稀薄 Top3 替代真实路径多段验证

## 研究目的

承接 [[R20260604-075_R010A8_过滤层稀薄状态分层只读发现]]。

A8 的只读结论是：

```text
action == A2
AND r010b3_filter_all_weak == true
AND top2_ret5 > 0
=> 用 Top3 替代原 A2 目标
```

这只是 close-to-close 局部反事实，不是组合真实路径。本轮 R010-A9 的任务是把该规则做成默认关闭策略参数，并在真实回测路径中验证收益、回撤和交易次数。

## 实现范围

本轮只允许作用在 `A2`，不扩展到小市值、不扩展到 A0/A1、不改变防守池。

## S2 真实路径结果

S2 规则：

```text
filter_all_weak
AND top2_ret5 > 0
=> Top3
```

| 2024 | 5 | +2.0747pct | 通过 |
| 2025-20260519 | 8 | +14.1801pct | 通过 |

S2 不能推广。失败主要来自 2022-06 至 2022-07 的宽反弹簇：当 `median_ret`、`ma20_breadth` 和 `ret20_breadth` 已经较高时，Top3 替代不再是低位修复，而是在强反弹里换到相对滞后的标的。

## S3 收窄规则

S3 只增加一个前置热度上限：

```text
filter_all_weak
AND top2_ret5 > 0
AND median_ret <= 0.03
=> Top3
```

含义：只在过滤层稀薄、Top2 近端转正、但池内中位 20 日收益尚未明显过热时替代。避免在广泛反弹已经展开后继续换到 Top3。
