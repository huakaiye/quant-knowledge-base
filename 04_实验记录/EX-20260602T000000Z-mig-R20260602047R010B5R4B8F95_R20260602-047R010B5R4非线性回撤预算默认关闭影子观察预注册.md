# R20260602-047 R010B5R4 非线性回撤预算默认关闭影子观察预注册

## 研究信念

关联信念：[[BLF20260523-003_波动率管理与条件降仓研究信念]]

当前相信的机制：

```text
条件降仓不应做全周期常开缩放。
非线性组合回撤预算只有在 B3 gate/tiered-v2 已确认风险窗口后，才可能作为二级仓位强度表达。
```

先验置信度：中高。

## 研究背景

R010-B5R3 四段参数敏感性与成本扰动已通过：

| 验证项 | 结果 |
| --- | --- |
| 参数邻域 | 6/6 相邻扰动优于 B3-gate-tiered-v2 |
| 成本扰动 | 2/2 同成本环境下候选仍有收益和回撤优势 |
| 新样本拖累 | 2025-20260519 最大拖累 -2.80pp，未超过 -5pp |
| 证据风险 | 5 个重复 summary 已记录到 duplicate_runs 审计文件 |

R010-B5R3 的决策边界为：

```text
promote_to_r010b5r4_shadow_observation_no_live
```

因此本轮不是上线，也不是回测参数继续优化，而是把下一阶段影子观察边界写清楚。

## 本次研究假设

主假设：

```text
在 B3 gate active 且 tiered-v2 已给出实际风险仓上限后，
非线性回撤预算可以作为只读影子建议，稳定给出可解释的 suggested cap，
并且不会改变真实 target weights、订单或实盘风险仓。
```

部分替代假设：

```text
若影子观察显示 nonlinear_suggested_cap 与 actual_tiered_cap 的差异稳定且误伤可控，
后续才允许预注册 R010-B5R5：
仅测试是否用非线性预算替代固定 cap70/cap60 的“仓位强度表达”。
不得替代 B3 gate 风险窗口识别。
不得替代 tiered-v2 strong/extreme 识别。
```

## 可观察预测

如果假设成立，预期应当观察到：

- 预测 1：日志能同时还原 `actual_tiered_cap`、`nonlinear_suggested_cap`、`cap_delta`、回撤状态和触发原因。
- 预测 2：观察期真实 `target_weights_actual`、订单和风险仓上限不因非线性影子建议改变。
- 预测 3：当 `nonlinear_suggested_cap < actual_tiered_cap` 时，后续 1/5/10 日误伤可解释，累计 log edge 不为负。
- 预测 4：当 `nonlinear_suggested_cap > actual_tiered_cap` 时，不削弱已由 tiered-v2 识别的强风险防守。

## 竞争性解释

| 竞争性解释 | 为什么可能成立 | 如何区分 |
| --- | --- | --- |
| 影子建议只是重复 tiered-v2 | 两者都依赖 B3 gate 风险窗口 | 统计 `cap_delta=0` 与非零差异比例 |
| 影子建议会误伤反弹 | 非线性回撤预算可能在回撤后继续压仓 | 对比后续 1/5/10 日风险资产收益 |
| 影子日志不完整 | 当前 R010-B5 真实路径日志偏应用型，不是 shadow-only 规格 | 先做字段审计，再决定是否补研究策略日志 |
| 观察字段改变交易 | 变量复用可能把 suggested cap 写入真实 cap | 逐日验证 target weights 和订单未改变 |
| 后验换参 | `d10_a5_f65` 四段更好 | 明确主观察参数仍为 `d10_a5_f70`，不得切换 |

## 观察边界

本研究只允许：

```text
记录非线性建议 cap。
记录实际 tiered cap。
记录二者差异和解释变量。
离线补充后验收益。
```

本研究禁止：

```text
修改 MiniQMT 默认开关。
修改真实 target weights。
修改订单逻辑。
绕过 B3 gate active。
绕过 tiered-v2 strong/extreme 识别。
把 R010-B5R4 解释为实盘上线。
```
