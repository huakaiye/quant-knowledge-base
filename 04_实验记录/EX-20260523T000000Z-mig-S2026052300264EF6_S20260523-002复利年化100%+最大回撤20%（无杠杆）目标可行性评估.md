# S20260523-002: 复利年化100% + 最大回撤20%（无杠杆）目标可行性评估

## 目标定义

- 本金10万，1年后达到20万（总收益100%）
- 复利年化100%，最大回撤≤20%
- **无杠杆**
- 本金小，可使用大基金无法使用的高频/小盘策略

## 数学基础

| 指标 | 值 |
|------|------|
| 所需日均收益 | 0.275% |
| 小市值做T实际日均 | 0.127%（占目标的46%）|
| 差距 | 2.2倍 |
| 所需年化夏普 | 2.91 |
| 小市值做T实际夏普 | 1.34 |
| 小市值做T盈亏比 | 1.01 |
| 小市值做T日胜率 | 55.6% |

## 现有策略分年表现（复利年化口径）

| 年份 | 复利年化 | 最大回撤 | 年化≥100%? | 回撤≤20%? | 双达标? |
|------|---------|---------|-----------|----------|--------|
| 2015 | +178.5% | -20.0% | ✓ | ✓ | **★** |
| 2016 | +45.7% | -17.2% | | ✓ | |
| 2017 | -17.0% | -25.8% | | | |
| 2018 | -13.8% | -37.4% | | | |
| 2019 | +19.2% | -36.9% | | | |
| 2020 | +1.1% | -24.3% | | | |
| 2021 | +63.4% | -18.5% | | ✓ | |
| 2022 | +56.9% | -14.7% | | ✓ | |
| 2023 | +26.3% | -14.1% | | ✓ | |
| 2024 | +29.8% | -30.8% | | | |
| 2025 | +58.5% | -25.6% | | | |

11年中仅2015年双达标。最好非牛市年份（2021）复利年化63%。

## 已验证的风控无法解决收益差距

R014中测试了约100种风控参数组合（止损式、波动率目标、趋势跟踪+止损），结论：
- 所有能有效压住回撤≤20%的规则，都会把收益压到年化<25%
- 根本矛盾：2017-2019持续1055天-37.4%回撤

## 理论上限

| 场景 | 复利年化 |
|------|---------|
| 原始策略 | 34.0% |
| 完美风控（所有亏损月空仓） | 60.8% |
| 完美风控+正收益月翻倍 | 147.4% |
| **目标** | **100%** |

完美风控（不可能的上限）也只有60.8%。纯靠小市值做T+风控无法达到100%。

## 核心结论

**必须从策略层面提升日均收益**，而非仅靠风控和仓位管理。

日均收益需要从0.127%提升到0.275%（+117%），路径：

### 方向1: 做T逻辑增强（最可行）
- **当前**：涨7.5%卖+回落5%确认，单一做T规则
- **可优化**：
  - 基于分钟级量价形态识别更精确的买卖点
  - 多层次做T（不只是一档买卖）
  - 波动率自适应阈值（高波放宽、低波收紧）
  - 参考社区实盘的高频涨停追击策略

### 方向2: 选股逻辑升级
- **当前**：ROE/ROA过滤+市值最小3只，每3天调仓
- **可优化**：
  - 集中持仓1-2只（10万不需要3只分散）
- 叠加动量质量因子（[[BLF20260523-002_平滑上涨软排序研究信念]]）
  - 涨停板强度分级筛选
  - 概念热度/资金流向选股

### 方向3: 叠加日内正交策略
- 集合竞价抢筹（9:15-9:25）
- 尾盘强势股埋伏（14:30-14:57）
- 涨停板龙头隔日追击
- 参考社区策略：9db.com年赛排名靠前的打板策略

### 方向4: 市场状态门控（解决回撤）
- 实现 [[R010-A_市场状态识别规格书]] 市场状态识别
- 熊市/高波期降仓或空仓
- 与策略增强结合：好年份赚更多、差年份亏更少

## 务实评估

| 方向 | 预期日均提升 | 实现难度 | 前置条件 |
|------|-----------|---------|---------|
| 做T逻辑增强 | +0.05-0.10% | 中 | 分钟级回测能力 |
| 集中持仓1-2只 | +0.03-0.05% | 低 | 回测验证 |
| 选股因子叠加 | +0.02-0.05% | 中 | 因子研究 |
| 涨停板追击 | +0.05-0.15% | 高 | 需新建策略 |
| 市场状态门控 | 降回撤为主 | 中 | R010-A实现 |

**最大不确定性**：社区中的「本金翻20倍」「年化96%」策略多数存在幸存者偏差和过度拟合。需要独立回测验证。

## 推荐的研究路线图

```
阶段1（1-2周）: 快速优化现有策略
├── 集中持仓1-2只回测（简单参数调整）
├── 做T阈值自适应回测（波幅联动）
└── 预期：日均收益提升到0.15-0.18%

阶段2（2-4周）: 叠加涨停板策略
├── 分析社区涨停追击策略的核心逻辑
├── 在平台中实现涨停板龙头追击策略
├── 回测验证真实收益
└── 预期：日均收益提升到0.20-0.25%

阶段3（并行）: 市场状态门控
├── 实现 [[R010-A_市场状态识别规格书]]
├── 叠加到策略中控制回撤
└── 预期：回撤从37%压到20-25%

阶段4（4-8周）: 组合验证
├── 最终策略组合回测
├── 目标验证：复利年化≥100% + 回撤≤20%
└── 如果未达标：调整目标或接受现实
```
