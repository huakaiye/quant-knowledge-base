# R20260603-056 R010B5S3 波动率状态与资金曲线修复组件诊断预注册

## 研究信念

关联信念：[[BLF20260523-003_波动率管理与条件降仓研究信念]]

R010-B5S2 已证明 T-1 组合级 `sigma_t` 与[[资金曲线状态]]诊断可以覆盖 27 个 R010-B5 counterfactual 事件，但没有 `sigma_target` 与 `vol_scale`，所以完整非线性仓位公式仍不可回测。

S3 的目标是固定分组诊断：在不扫描参数、不生成仓位、不运行回测的前提下，检查波动率状态和资金曲线修复状态是否能解释既有 counterfactual edge。

先验置信度：中低。

## 本次研究假设

如果波动率项或资金曲线状态有组件级价值，应当观察到：

- 自然波动率分组，如 `sigma20 > sigma60` 或 20/60 短长波动率固定 0.90/1.10 分组，在 5日与10日 edge 上同时偏正。
- 资金曲线修复状态在样本数足够时，不只是 1日偏正，而是 5日和10日也稳定偏正。
- 负控制不能出现同等或更强的正向表现。
- 低 `median_ret` 固定阈值只能作为 parked reference，不能绕过 R5R4 的独立样本外限制。

## 预注册固定规则

| 类别 | 固定规则 |
| --- | --- |
| 样本 | 固定使用 S2 27 个 R010-B5 counterfactual 事件 |
| 观察周期 | 1日、5日、10日，1日不能单独支持结论 |
| 最小样本 | `event_count>=8` 且 `segment_count>=2` 才能进入 watch 判断 |
| watch 条件 | 5日与10日均值为正，且 5日与10日正率均不低于 60% |
| 波动率自然分组 | `sigma_t20_ann_t1 > sigma_t60_ann_t1` 与 `<=` |
| 短长波动率分组 | `sigma20/sigma60 <= 0.90`、`0.90-1.10`、`>=1.10` |
| 资金曲线状态 | 沿用 S2 的 `capital_curve_state_t1` |
| 资金修复状态 | `capital_curve_ret5_t1 > 0` 与 `<=0` |
| 回撤分档 | T-1 回撤固定 10%、15%、20% 分档 |
| 参考项 | `risk_scale == 0.70`、`actual_tiered_cap`、Top1 斜率符号、`ret20_breadth>=0.50` |
| parked reference | 低 `median_ret<=-0.015/-0.01`；`<=-0.0141` 仍为后验禁止参考 |
| 负控制 | 标签旋转 7 位、事件序号奇偶、日期哈希奇偶 |

## 竞争性解释

| 竞争性解释 | 如何区分 |
| --- | --- |
| 波动率状态真正有组件价值 | 自然波动率分组通过 watch，且负控制不通过 |
| 资金曲线修复状态有补仓价值 | 修复组样本数足够且 5/10 日同步为正 |
| 低 `median_ret` 仍是主要线索 | 固定阈值仍只能作为 parked reference，必须等待独立样本外 |
| 任意分组都能切出好结果 | 负控制若同样偏正，则降级解释 |
