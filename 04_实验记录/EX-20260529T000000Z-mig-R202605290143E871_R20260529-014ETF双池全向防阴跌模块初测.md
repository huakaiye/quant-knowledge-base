# R20260529-014 ETF双池全向防阴跌模块初测

## 研究信念

关联信念：[[BLF20260529-006_ETF阴跌防守与动量轮动研究信念]]

当前相信的机制：ETF 动量轮动在部分慢跌或风险偏好收缩阶段会继续追逐局部强势标的，但这些标的可能已经处于高位脆弱状态；如果能及时识别阴跌状态，应当减少错误追强或切入防御资产。

先验置信度：中

## 本次研究假设

在 [[ETF双池动量轮动]] 上加入“分资产桶阴跌防守”后，策略应减少 2022 年弱势环境下的回撤，并且不显著牺牲年度收益。

## 可观察预测

如果假设成立，预期应当观察到：

- 预测 1：2022 年全向防阴跌版本的最大回撤低于原始双池。
- 预测 2：防御 ETF 不应长期占用资金；市场恢复时应能重新进入风险 ETF。
- 预测 3：软惩罚版本应比硬过滤版本更少错过强势资产，收益不应显著弱于硬过滤。

## 竞争性解释

| 竞争性解释 | 为什么可能成立 | 如何区分 |
| --- | --- | --- |
| 防御仓切换缺陷 | 防御 ETF 如果不卖出，会导致恢复进攻失败 | 检查 511880 是否与风险 ETF 长期共存 |
| 资产桶代理失真 | 用原油 ETF 代表能源资源，不能代表煤炭 ETF 的风险 | 检查最大回撤期间持仓和对应资产桶基准 |
| 规则过度触发 | 多个资产桶频繁进入阴跌，候选池被过度压缩 | 统计激活桶天数和累计过滤数量 |
| 动量自身脆弱 | Top1 虽强但已过热，阴跌桶信号无法识别个体衰竭 | 对持仓自身做回撤、短动量和新高距离审计 |

## 改动内容

本轮实现并测试了三个版本：

- 全向防阴跌硬过滤：每个资产桶有独立风险基准，进入阴跌后剔除桶内 ETF。
- 防御仓退出修复：当重新出现进攻目标时，卖出 `511880.XSHG`，避免防御仓锁死资金。
- 全向防阴跌软惩罚：不剔除阴跌桶 ETF，只将其动量得分乘以 `0.62`。
- A股限定防阴跌：只对 A股核心、A股中小盘、A股成长、A股综合四个风险桶启用阴跌过滤，港股、美股、商品和债券不触发该模块。

## 偏差诊断

最大意外：全向阴跌规则大量触发，但几乎没有降低最大回撤。

观察到的触发强度：

| 指标 | 硬过滤修复版 |
| --- | ---: |
| 阴跌过滤事件 | 145 |
| 累计剔除候选 | 8737 |
| A股成长激活近似天数 | 59 |
| 美股科技激活近似天数 | 80 |
| 美股宽基激活近似天数 | 53 |
| A股核心激活近似天数 | 41 |
| 能源资源激活近似天数 | 31 |

A股限定版触发更克制，但依然没有改善回撤：

| 指标 | A股限定硬过滤 |
| --- | ---: |
| 阴跌过滤事件 | 89 |
| 累计剔除候选 | 6699 |
| A股成长进入/退出次数 | 4 / 4 |
| A股中小盘进入/退出次数 | 4 / 4 |
| A股核心进入/退出次数 | 3 / 3 |
| A股综合进入/退出次数 | 3 / 3 |
| 最大回撤日期 | 2022-08-31 |
| 最大回撤前关键买入 | 2022-08-25 买入 `515220.SH` |

主要原因：

- 机制偏差：阴跌状态按资产桶基准判断，但实际回撤可能来自桶内个体，例如 `515220.XSHG` 煤炭 ETF。
- 信号偏差：能源资源桶使用 `501018.XSHG` 原油作为基准，无法代表煤炭、有色、钢铁等子行业。
- 组合暴露：单仓 Top1 策略即便过滤部分桶，也会继续满仓追逐另一个高波动强势标的。
- 风控偏差：8% 成本止损偏慢，不能处理高位强势 ETF 的动力衰竭。
- 适用边界：A股限定减少了跨资产误杀，但没有处理商品资源和个体 ETF 动量衰竭，因此不能简单得出“阴跌只适合 A股且可直接采纳”。

## 下一轮实验

下一轮实验目的：把“整桶阴跌判断”拆成状态日志和更细的交易门控。

最有信息量的下一步：

- 做 S1 状态日志：不改变交易，只记录候选 Top5 的资产桶、绝对动量、短动量、新高距离、近高回撤。
- 做 S2 状态归因：比较阴跌桶、非阴跌桶、Top1 动力衰竭状态下的次日和五日收益。
- 做 S3 交易消融：只有当 Top5 都低于绝对动量阈值，或 Top1 自身触发高位回落，才进入防守。
- 拆分能源资源桶：至少区分原油、煤炭、有色，避免单一基准代理整桶。
- 做 A股限定的样本外复核，但优先级低于 Top5 动力衰竭日志，因为 2022 结果没有表现出回撤改善。

## 结论

结论状态：revise

是否进入策略路线图：暂不进入实盘路线图。当前阴跌模块只能作为研究候选，不能直接并入 [[ETF双池动量轮动]]。A股限定版本说明“全资产机械推广阴跌”不可靠，但并未证明 A股阴跌过滤本身足够有效。下一轮应优先验证“候选自身动力衰竭”和“Top5整体无强势”是否比资产桶硬过滤更有效。

## 风险备注

本轮只有 2022 单年窗口，尚未做 2020-2026 全周期、分年样本外、参数扰动、延迟一日执行、溢价率过滤和滑点放大。所有结论只用于指导下一轮研究，不构成采纳依据。
