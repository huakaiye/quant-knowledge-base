# R20260601-029 R010B4 B3Gate强度分层TieredV2预注册

## 研究信念

关联信念：[[BLF20260529-006_ETF阴跌防守与动量轮动研究信念]]

R010-B4 四段验证显示：[[B3 gate]] 后更强降仓在冲击窗口和弱势段确有价值，但固定 cap70/cap60 会在部分 gate 日误伤，tiered-v1 又过窄且不稳定。因此，下一步不应继续固定强度，而应把“是否进入更强防守”建立在更持久、更内生的弱化证据上。

## 本次研究假设

在 `[[A2-slope004]] + B3-gate-median-or-breadth-cap80` 基础上，只有当 B3 gate active 同时满足“全弱持续已经足够久、Top5 滤波衰竭充分、池内中位收益不强”时，才把普通 cap80 提升到 cap70；在此基础上，若中位收益更差或广度更低，再升级到 cap60。

## 预注册规则

首选 [[tiered-v2（分层强度防守）]] 规则：

```text
普通 gate：
  B3 gate active，且不满足 strong_risk
  -> cap80

strong_risk：
  B3 gate active
  AND r010b3_all_weak_cum_days >= 10
  AND r010b3_all_weak_persistence_gate_median_ret <= 0
  AND r010b3_top5_filter_exhausted_count >= 4
  -> cap70

extreme_risk：
  strong_risk
  AND (
    r010b3_all_weak_persistence_gate_median_ret <= -0.015
    OR r010b3_all_weak_persistence_gate_ma20_breadth <= 0.25
  )
  -> cap60
```

备选观察规则：

```text
precision_extreme：
  strong_risk
  AND top1_slope10 <= 0.00352
  -> cap60
```

备选规则不作为首轮回测主规则，原因是它能提高动作日胜率，但会漏掉 2025-04-07 这类 Top1 斜率仍高、但强防守同日收益很高的冲击日。

## 可观察预测

- 预测 1：tiered-v2 首选规则的 2024 smoke 应优于 B3-gate-cap80，最大回撤至少改善 1.5pp，复合收益不低于 cap80 的 98%。
- 预测 2：四段真实路径中，tiered-v2 不应出现 2020-2021 类固定强度拖累；若 2020-2021 收益相对 cap80 低于 -1pp 且回撤不改善，则 revise。
- 预测 3：动作日胜率整体应不低于 75%，且任一有触发分段不低于 70%。
- 预测 4：2025-04 冲击窗口应保留相对 cap80 的主要回撤改善；若因分层过窄漏掉该窗口，需回退到强度边界研究。
- 预测 5：cap60 天数应显著少于固定 cap60，且误伤日数量低于固定 cap60。

## 竞争性解释

| 竞争性解释 | 为什么可能成立 | 如何区分 |
| --- | --- | --- |
| 动作日重建过于乐观 | 当前审计按既有 cap70/cap60 净值逐日重建，未模拟持仓路径变化 | 必须运行真实策略路径回测，不能只看动作日 edge |
| Top5 衰竭只是与 gate 重复 | `r010b3_top5_filter_exhausted_count>=4` 可能只是在重复[[B3全弱信号]] | 对比 v1、固定 cap60、strong_risk、extreme_risk 的触发天数和分段表现 |
| 低 Top1 斜率过滤过窄 | `top1_slope10<=0.00352` 会提高精度但漏掉冲击日 | 只作为备选，不做首轮主规则 |
| 持续全弱天数过拟合 | `cum_days>=10` 来自 limited2/gate 既有研究，可能错过早期冲击 | 2024-06 起点和 2025-04 冲击窗口必须单独审计 |

## 证伪测试与决策规则

| 测试 | 目的 | kill / revise / promote 规则 |
| --- | --- | --- |
| 配置审计 | 确认只新增默认关闭参数，未改变 cap80 主候选 | 若影响默认配置或混入 R026 交易动作，kill |
| WSL dry-run | 确认配置可执行且字段当日可得 | 若字段缺失或依赖未来窗口，kill |
