# 研究质量审计规范

本规范用于约束 Agent 像研究员一样工作，核心目标是避免未来函数、过拟合、后验调参和不可复现实验。

## 硬性原则

任何策略改进在没有完成本页审计前，只能标记为 `observe` 或 `draft`，不得进入 `promote`。

## 未来函数审计

未来函数是指策略在某个时间点使用了当时无法知道的信息。发现未来函数后，相关回测结果不能作为有效证据。

### 必查项目

| 项目 | 检查问题 | 通过要求 |
| --- | --- | --- |
| 数据时间戳 | 因子、价格、财务、成分、指数数据是否只使用交易时点前已知信息 | 明确数据可得时间，不用事后修订值 |
| 信号滞后 | 今日信号是否用了今日收盘后才知道的数据 | 日频信号默认至少滞后一根 bar，除非证明盘中可得 |
| 交易价格 | 下单价格是否在信号生成后可成交 | 不用同一根 bar 的未来价格成交 |
| 股票池/ETF池 | 成分池是否使用了未来才知道的存续、退市、入选信息 | 使用当期可知的池，或明确说明静态池限制 |
| 财务数据 | 财报、估值、基本面字段是否按公告日可得 | 不按报告期直接回填 |
| 数据修订 | 宏观、指数成分、复权、停牌等是否可能后验修订 | 记录数据源口径和修订风险 |
| 聚宽兼容 | 平台实现是否与聚宽语义对齐 | 关键分叉需有日志、probe 或对比证据 |
| Shadow 标签 | 默认关闭或观察信号是否被误当成可交易信号 | Shadow 只能观察，不能直接进入默认策略 |

### 必做负控

至少选择一类负控：

- 将信号整体向后错位，检查收益是否异常保留。
- 使用随机打乱的信号替代目标信号。
- 删除目标模块做消融。
- 对关键触发日读取 `current_dt`、价格和数据窗口日志。

如果负控仍能产生相近收益，优先怀疑未来函数、样本泄漏或过拟合。

## 过拟合审计

过拟合是指策略只适合已经看过的样本，不具备可迁移机制。

### 必查项目

| 项目 | 检查问题 | 通过要求 |
| --- | --- | --- |
| 实验预注册 | 是否先写假设、预测、参数范围和失败条件 | 结果出来前已有实验记录草稿 |
| 样本划分 | 是否区分样本内、验证集、样本外和压力期 | 不能只在全样本看最优结果 |
| 参数范围 | 参数网格是否提前定义 | 不允许看结果后无限扩网格 |
| 敏感性 | 策略是否只在单点参数有效 | 邻近参数应表现合理 |
| 多重比较 | 是否跑了大量组合却只报告最好一个 | 必须报告搜索空间和失败样本 |
| 消融实验 | 收益是否来自目标模块 | 删除模块后应有可解释变化 |
| 成本扰动 | 是否考虑佣金、滑点、冲击和换手 | 成本加倍或滑点扰动后仍有合理边际 |
| 样本外 | 未参与设计的区间是否仍支持假设 | 样本外不通过不得 promote |
| 机制一致性 | 指标改善是否符合机制预测 | 不能只用收益结果倒推理由 |

## 证据等级

| 等级 | 名称 | 含义 | 可做决策 |
| --- | --- | --- | --- |
| L0 | 灵感 | 文献、观察或想法，未实验 | 只能进入候选池 |
| L1 | 探索回测 | 已跑结果，但无完整预注册或审计 | 只能 `observe` |
| L2 | 预注册回测 | 有假设、预测、基准和证伪条件 | 可 `promote_candidate` |
| L3 | 稳健性验证 | 有样本外、敏感性、成本和消融 | 可 `promote` 到更高验证 |
| L4 | 影子观察 | 默认关闭，真实时间顺序观察 | 可考虑候选合并 |
| L5 | 生产候选 | 工程、风控、实盘边界均通过 | 仍需人工最终确认 |

默认规则：

- L0/L1 不允许改变当前最佳策略。
- L2 只能成为候选。
- L3 以上才允许写入方向页的“有效证据”。
- L4/L5 涉及真实交易边界，必须由主控或人工最终确认。

## Agent 审计职责

| Agent 层级 | 可以做 | 不可以做 |
| --- | --- | --- |
| 子查 | 检查字段缺失、路径存在、是否有样本外和成本字段 | 判断无未来函数 |
| 子研 | 归纳支持证据、反对证据和过拟合风险 | 独立 promote |
| 子执 | 按预注册配置运行回测和输出日志 | 改参数追结果 |
| 主控 | 最终判断未来函数、过拟合和决策状态 | 省略反证 |

## 实验记录强制字段

每份正式实验必须包含：

- 未来函数审计。
- 过拟合审计。
- 样本划分。
- 参数搜索空间。
- 成本和换手检查。
- 消融或负控。
- 证据等级。

缺任一项时，实验状态不得高于 `promote_candidate`。
