# 文献归档规范

本库需要长期从经济学、金融学、资产定价、市场微观结构、行为金融、宏观金融、计量和机器学习文献中获取研究灵感。

## 什么时候归档

只要一篇文献可能影响策略研究，就应建立文献卡片。不要等读完全文才归档。

| 阶段 | 动作 |
| --- | --- |
| 只看到题名或摘要 | 放入 `06_文献资料/00_待处理/`，状态 `todo` |
| 已读摘要和结论 | 建立 `LIT` 文献卡，状态 `skimmed` |
| 已读方法和数据 | 补充识别策略、数据、变量、可复现性，状态 `read` |
| 产生因子或数据灵感 | 新建 `FAC` 或 `DATA` 卡，并反向链接文献 |
| 进入实验 | 在实验记录中引用 `lit_id` 或灵感卡 ID |
| 已转化或暂不使用 | 状态改为 `archived` 或 `parked` |

## 分类目录

| 目录 | 收纳内容 |
| --- | --- |
| `00_待处理/` | 未读或只记录线索 |
| `01_经济学顶刊/` | AER、QJE、JPE、Econometrica、Review of Economic Studies 等 |
| `02_金融学顶刊/` | Journal of Finance、JFE、Review of Financial Studies 等 |
| `03_资产定价/` | 因子、风险溢价、横截面收益 |
| `04_市场微观结构/` | 交易、流动性、订单簿、冲击成本 |
| `05_行为金融/` | 投资者行为、过度反应、注意力、情绪 |
| `06_宏观金融/` | 宏观变量、政策、利率、通胀、信用 |
| `07_计量与机器学习/` | 识别、预测、稳健性、模型验证 |
| `08_已归档/` | 已转化、已否定或暂不使用 |

## 文献卡必须包含

- `lit_id`
- 标题
- 作者
- 年份
- 期刊或来源
- DOI 或 URL
- 研究问题
- 使用的数据
- 识别方法或模型
- 核心结论
- 可转化的因子、数据、机制或模块
- 对当前策略的可能影响
- 局限和不能直接外推的地方

## 版权边界

文献卡写中文摘要和研究解释，不粘贴长篇原文。论文 PDF 默认不入 Git。若本地保存原文，放在被 `.gitignore` 忽略的 `原文_本地不入库/`，并在文献卡中登记本地路径。

## 从文献到实验的连接

文献不能直接跳到策略结论，必须经过至少一个中间资产：

```text
LIT 文献卡 -> FAC 因子卡 / DATA 数据卡 / MECH 机制卡 / IDEA 模块卡 -> EX 实验记录
```

这样后续才能分辨：结果来自文献机制、数据代理、实现细节，还是单纯参数拟合。
