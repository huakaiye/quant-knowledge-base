# D20260602-B3-gate 误拦截 Top1 斜率放松误伤归因完成

## 决策摘要

Top1 低斜率放松的 2025-20260519 误伤归因已完成。结论是：低 Top1 斜率在该段更像趋势修复期的滞后弱斜率标签，而不是足够干净的提前风险信号。

当前判断为 `revise_to_watch_label_no_new_guardrail`：保留为只读观察标签，不再基于本轮后验归因继续拼新 guardrail，也不进入实盘映射。

## 是什么

本节点解释上一节点 [[D20260602-B3-gate误拦截Top1斜率放松四段压力测试完成]] 中的 2025-20260519 反证。

被审计规则仍是：

```text
B3 raw 已触发
AND 原 median/breadth gate blocked
AND top1_slope10 <= 0.00352
```

它的问题不在于缺少一个简单阈值，而在于低 Top1 斜率会在部分修复期把原 gate 正确拒绝的状态重新打开。

## 相比上个节点

上一节点已经说明四段不晋级。本节点进一步判断后续路线：不再追加后验 guardrail 组合，不再为该规则做交易化补丁，只把它保留为 B3 gate 默认关闭观察方案里的次级误拦截标签。

## 当前判断

```text
revise_to_watch_label_no_new_guardrail
```

允许：
- 保留归因审计产物。
- 在默认关闭观察中记录 `top1_slope10<=0.00352` 作为解释字段。
- 等未来样本验证它是否再次出现在真实 blocked false negative 之前。

禁止：
- 把 Top1 低斜率放松接入实盘默认开关。
- 用本轮后验 guardrail 继续拼交易规则。
- 替代 `[[median_ret（中位收益）]]<=0 OR [[ma20_breadth（广度）]]<=0.40` 主 gate。
- 与 R026 叠加成新的仓位动作。

## 决策

```text
revise_to_watch_label_no_new_guardrail
```

解释：Top1 低斜率放松有解释价值，但交易价值不足。主 gate 的 `median_ret` 与 `ma20_breadth` 过滤仍应保持优先级，低斜率只能作为观察字段。

3. 下一条策略改进应回到更高优先级主线：[[tiered-v2（分层强度防守）]] shadow 观察边界、[[B3 gate]] 默认关闭观察，或其他未证伪的低权限离线研究。
