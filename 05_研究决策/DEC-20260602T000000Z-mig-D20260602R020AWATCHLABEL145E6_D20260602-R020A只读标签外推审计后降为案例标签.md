# D20260602-R020A 只读标签外推审计后降为案例标签

## 决策摘要

R020A 综合只读标签完成独立窗口外推审计。它能解释部分长趋势回撤，但外推稳定性不足：2022-10 完全漏掉，2024-09 谷底前 T-10 不命中。因此决策为 `park_case_label_not_generalizable_trigger`。

## 是什么

本节点复用 [[R20260602-030_R020A_2021慢性失速只读标签审计]] 的同一组预注册标签，检查 2022-10、2023-12、2024-09 等独立深回撤窗口。目标是判断 2021-05 的慢性失速解释是否可外推为通用触发器。

平台脚本：

```text
scripts/research/analyze_r020a_watch_label_oos_windows.py
```

输出目录：

```text
results/v2/research/R020/2021_gap_watch_label_oos_windows/
```

本节点不启动回测、不访问数据库、不改实盘配置。

## 相比上个节点

上一节点 [[D20260602-R020A-2021慢性失速只读标签审计完成]] 的判断是：R020A 综合标签能解释 2021-05，但没有交易型 edge，只能观察。

本节点把它放到独立窗口外推。结果显示该标签不能稳定覆盖 2022-10、2023-12、2024-09 三类独立窗口，尤其是 T-10 稳定性不足。因此路线从 `observe_readonly_explanation_no_trade_edge` 收缩到 `park_case_label_not_generalizable_trigger`。

## 当前判断

```text
park_case_label_not_generalizable_trigger
```

允许：

- 保留 `r020_2021_gap_composite_watch` 作为 2021-05 和部分长趋势回撤的解释字段。
- 在未来归因中引用它解释“权益失血但主 gate 未触发”的案例。

禁止：

- 不把 R020A 综合标签并入 B3 gate。
- 不基于该标签新增 cap70、cap60 或清仓动作。
- 不继续沿该标签拼后验阈值。

| 2024-09 独立窗口 T-20 命中 2 天、T-10 命中 0 天 | 窗口早期有解释力，但谷底附近不稳定 |
| 独立窗口 T-10 命中率 1/3 | 不足以作为通用触发器 |
| 上一轮后 20 日表现无交易 edge | 即使命中，也不能证明适合降仓 |

## 反对证据

park_case_label_not_generalizable_trigger
