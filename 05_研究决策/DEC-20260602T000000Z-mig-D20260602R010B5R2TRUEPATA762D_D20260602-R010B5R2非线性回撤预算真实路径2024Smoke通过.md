# D20260602-R010B5R2 非线性回撤预算真实路径2024 Smoke通过

## 决策摘要

R010-B5R2 把 R010-B5R1 的非线性组合回撤预算从净值层代理推进到真实策略路径。2024 smoke 结论很明确：

```text
overlay_tiered_v2_nonlinear 通过，可以进入四段真实路径验证。
replace_tiered_fixed_caps_nonlinear 失败，不能替代 tiered-v2 固定 cap70/cap60 强度层。
本节点仍然不实盘。
```

当前决策：

```text
promote_overlay_to_four_segment_kill_replacement_no_live
```

## 是什么

R010-B5R2 是组合层二级风控研究。它不替代 A2、B3 gate 或 B3-gate-tiered-v2，而是在 B3 gate 和 tiered-v2 已给出风险仓上限之后，再根据组合自身回撤做连续缩放。

本轮真实路径固定候选为：

```text
nl_endo_gate_any_d10_a5_f70
账户自身回撤驱动
10% 回撤后指数衰减
最低缩放 70%
只在 normal / strong / extreme 风险窗口内生效
```

## 相比上个节点改变了什么

上个节点 R010-B5R1 的结论是 `promote_proxy_candidate_for_true_path_no_live`，证据只来自净值层代理。

本节点做了三处升级：

- 进入真实策略路径，使用研究策略文件、真实调仓事件、真实订单与持仓输出。
- 同时测试“叠加 tiered-v2”和“替代 tiered 固定强度层”两条路径。
- 加入默认关闭、状态更新、下单前接入点和 14:59 权益同步的时点审计。

## 当前判断

```text
promote_overlay_to_four_segment_kill_replacement_no_live
```

可以做：

- 继续补跑 overlay 版本的四段真实路径。
- 汇总四段收益、最大回撤、交易笔数、动作日和关键回撤窗口。
- 保留 replacement 失败结果，作为“不替代固定强度层”的反证。

不能做：

- 不能修改 MiniQMT。
- 不能启用非线性实盘开关。
- 不能替代 B3 gate。
- 不能替代 B3-gate-tiered-v2 的固定 cap70/cap60 层。
- 不能继续大规模参数扫描。

## 支持证据

| 证据 | 结果 |
| --- | ---: |
| B3-gate-tiered-v2 2024复合收益 | +67.01% |
| overlay 2024复合收益 | +71.57% |
| overlay 相对收益 | +4.57pp |
| B3-gate-tiered-v2 2024最大回撤 | -26.99% |
| overlay 2024最大回撤 | -24.63% |
| overlay 回撤改善 | +2.36pp |
| overlay 2024主回撤窗口改善 | +2.40pp |
| overlay 交易笔数变化 | 0 |
| overlay 实际缩放日 | 12 |

这些证据说明：非线性预算作为 tiered-v2 之上的二级缩放层，在 2024 真实路径中没有重现 R010-B5 的长期低仓锁定问题，并且在主回撤窗口贡献明确。

## 反对证据

| 证据 | 说明 |
| --- | --- |
| replacement 收益低于基准 | 2024 复合收益 +64.71%，低于 tiered-v2 的 +67.01% |
| replacement 回撤差于基准 | 最大回撤 -28.19%，差于 tiered-v2 的 -26.99% |
| replacement 主回撤窗口更差 | 主回撤窗口改善 -0.79pp |
| 改善集中在2024主回撤 | 第二回撤和年末回撤几乎没有改善 |
| 样本仍是单段 smoke | 尚未验证 2020-2021、2022-2023、2025-20260519 |

关键反证含义：

```text
非线性组合回撤预算不能替代 tiered-v2 的强度识别。
median_ret、ma20_breadth、cum_days 和 top5 exhausted 仍然是必要的风险强度信息。
```

## 决策边界

本节点不是实盘升级。当前实盘边界仍为：

```text
A2-slope004 + B3-gate-tiered-v2
```

R010-B5R2 当前只能作为离线研究分支，后续最低要求：

```text
overlay 四段真实路径至少 3/4 不明显劣于 tiered-v2。
复合收益保留率不低于 95%。
最大回撤相对 tiered-v2 有稳定改善。
2025-20260519 不出现灾难性拖累。
动作日质量不得明显低于既有 tiered-v2。
```
