## 决策结论

`revise`

A23 92/4/d09 不直接进入换手约束 formal，也不替代当前默认 hard5。下一步先做 hard5、A22 cap70、A23 92/4/d09 的同成本配对 A/B；只有同成本下仍能通过收益和回撤门槛，才允许继续研究持仓延续或换手约束版本。

## 这个节点是什么

这是 A23 从“看起来能替代 hard5 的软预算候选”转为“需要先证明同成本抗压能力”的决策点。

通俗说，A23 的想法是：高分资产不一定全禁，而是给它较小仓位预算。但 VX4J 发现，A23 相对 A22 的独有贡献非常窄，主要来自 2020 年 3 次 broad-blowoff 降仓；2025/2026 的强收益主要来自 A22/A23 共同的 base cap70，不是 A23 分层规则独有。

## 相比上一节点改变了什么

- 原先：A23 是 hard5 替代研究的首选 soft budget 候选，下一步可能直接做成本/换手约束版本。
- 现在：A23 仍有研究价值，但不能直接做新机制 formal；必须先做同成本配对，避免把“高成本下普遍变弱”误读成“A23 需要少换手就能修复”。

## 支持证据

- VX4J 只读脚本通过 `py_compile`，输出 `summary.json` 标记 `read_only=true`、`backtest_executed=false`、`data_update=false`。
- A23 candidate 相对 A22 cap70 只在 2020_2021 final 多 `634.92`，其余三段差异为 `0`。
- 2020 的差异主要集中在 3 个 broad-blowoff 触发窗口，H10 增量合计约 `+774.60`，支持“2020 风险削减”解释。
- 2025_20260519 的强收益来自 A22/A23 共同的 base cap70：A22 相对 hard5 的 34 个 base 事件 H5 增量合计 `+48297.75`，不是 A23 tier 独有。
- `cost2x_slip2bps` 四段均弱于普通成本 A23，但这不是公平同成本比较，不能直接证明 A23 本身在高成本下弱于 A22。

## 反对证据

- A23 独有样本太少：broad-blowoff 只在 2020 出现 3 次。
- 2022_2023、2024、2025_20260519 中 A23 与 A22 cap70 完全相同，说明状态分层本身没有额外贡献。
- 高成本版本和普通成本版本路径高度重合但不同序，说明成本会反馈到现金、股数和后续路径，不能只用费用差解释。
- 直接做换手约束可能重复 TC8U/C35J 的问题：少交易看起来改善，但可能只是任意延迟造成的负控伪改善。

## 边界

本决策不说明 A23 无效，只说明 A23 不能直接晋级，也不能马上进入换手约束 formal。A22/A23 的 soft budget 方向仍优于简单“取消 hard5”或 Top4、多周期确认等失败路线。

默认策略继续保留 hard5；A22 cap70 和 A23 92/4/d09 保留为研究候选。

## 后续动作

- 新开同成本配对实验：`hard5_cost2x_slip2bps`、`a22_cap70_cost2x_slip2bps`、`a23_92_4_d09_cost2x_slip2bps`。
- 不扩 A23 阈值网格，不先做换手约束 formal。
