# D20260602-B3-gate 误拦截 Top1 斜率放松四段压力测试完成

## 决策摘要

Top1 低斜率放松完成 2020-2021、2022-2023、2024、2025-20260519 四段压力测试。整体复合收益相对 B3 gate 小幅提高，但 2025-20260519 出现收益拖累且最大回撤没有改善，因此当前判断为 `observe_not_promote_history_pressure_pass`。

本节点不改变当前主候选，不允许开启实盘默认开关。

## 是什么

规则仍是一个窄放松条件：

```text
B3 raw 已触发
AND 原 median/breadth gate blocked
AND top1_slope10 <= 0.00352
```

若成立，则允许沿用 B3 gate 的 limited2 cap80。它只修正原 gate blocked raw 的一小类误拦截，不改 B3 raw、不改 cap80、不引入 R026。

## 相比上个节点

上一节点：[[D20260601-B3-gate误拦截Top1斜率放松2024压力测试通过]]

上一节点只证明 2024 单段小幅通过。本节点补齐三段配置、配置审计、WSL dry-run、正式回测和四段汇总，发现该规则不是稳定晋级候选：它在前三段小幅改善，但在 2025-20260519 明显拖累收益。

## 当前判断

```text
observe_not_promote_history_pressure_pass
```

允许：
- 保留四段压力测试结果。
- 后续只做误伤归因、默认关闭观察或未来样本复核。

禁止：
- 替代当前 B3 gate 主候选。
- 进入 MiniQMT 默认开启。
- 与 R026 叠加成交易开关。
- 因四段复合收益小幅改善而 promote。

## 反证

| 反证 | 说明 |
| --- | --- |
| 2025-20260519 收益拖累明显 | 相对 B3 gate 收益 -2.4583pp，最大回撤还小幅恶化 -0.0023pp |
| 2025 active 日后验偏负 | 2025-03-04、2025-05-23、2025-05-26、2025-06-23 四个 active 日 T+10 净值差均为负 |
| 改善幅度过小 | 最差回撤改善只有 +0.0544pp，不能支撑交易化 |
| 阈值仍来自历史未来标签 | `top1_slope10<=0.00352` 不是事前理论阈值 |
| relax pass 太多 | 四段 relax pass 合计 234 天，但 active 只有 18 天，说明标记层噪音仍高 |

## 决策

```text
observe_not_promote_history_pressure_pass
```

解释：该规则在历史四段没有达到 kill 条件，因为只有一个分段收益拖累且无回撤改善；但它也没有达到晋级条件，因为收益改善很小、回撤改善极小，且 2025-20260519 给出清晰反证。

## 下一步

1. 不再推进实盘映射或默认开关。
2. 若继续研究，只做 2025-20260519 误伤归因，重点解释 2025-03、2025-05、2025-06 的 active 日为什么错过后续修复。
3. 该规则保留为 B3 gate 默认关闭观察页下的次级误拦截标签，不替代 `median_ret<=0 OR ma20_breadth<=0.40` 主 gate。
