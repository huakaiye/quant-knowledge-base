# D20260602-R010B5R2 非线性回撤预算四段真实路径验证通过

## 决策摘要

R010-B5R2 的叠加版 `overlay_tiered_v2_nonlinear` 已完成四段真实路径验证。结论是：

```text
promote_offline_candidate_no_live
```

含义：

```text
非线性组合回撤预算可以作为 B3-gate-tiered-v2 之后的离线候选继续推进。
它不是实盘启用结论。
它不能替代 B3 gate，也不能替代 tiered-v2 的固定强度分层。
```

## 是什么

本节点验证的是固定候选：

```text
nl_endo_gate_any_d10_a5_f70
reference_mode = endogenous
drawdown_threshold = 10%
decay = 5
floor = 70%
eligible_levels = normal / strong / extreme
apply_mode = post_cap_multiplier
```

策略语义：

```text
先由 B3 gate 与 tiered-v2 决定风险仓上限。
若处在 B3/tiered 风险窗口，再根据组合自身回撤做指数型连续缩放。
回撤越深，风险仓上限越低；回撤修复后，缩放自然回升。
```

## 相比上个节点改变了什么

上个节点只完成 2024 smoke，并且同时测试了叠加版与替代版。结论是叠加版可继续，替代版停止。

本节点新增：

- 补齐 2020-2021、2022-2023、2025-20260519 三段真实路径。
- 用四段拼接口径比较复合收益、最大回撤和交易笔数。
- 重建 27 个非线性动作日的胜率、平均超额和累计 log edge。
- 检查 2021-05、2024 主回撤、2025-04 等关键窗口。

## 当前判断

```text
promote_offline_candidate_no_live
```

可以做：

- 新建 R010-B5R3 参数敏感性与成本扰动预注册。
- 固定少量变体检验 `threshold / decay / floor` 的稳定性。
- 设计默认关闭的影子观察边界。

不能做：

- 不能修改 MiniQMT。
- 不能把 R010-B5R2 加进实盘默认开关。
- 不能替代 B3 gate。
- 不能替代 B3-gate-tiered-v2 固定 cap70/cap60 强度层。
- 不能大规模后验参数扫描。

## 决策边界

当前实盘边界仍然是：

```text
A2-slope004 + B3-gate-tiered-v2
```

R010-B5R2 当前只升级为离线候选。进入任何实盘相关动作前，至少还需要：

```text

成本扰动通过。
未来函数和状态时点复核通过。
影子观察边界和停止条件写入研究库。

```
