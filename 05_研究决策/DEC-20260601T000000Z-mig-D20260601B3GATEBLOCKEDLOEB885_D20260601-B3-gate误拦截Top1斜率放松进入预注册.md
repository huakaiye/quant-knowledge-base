# D20260601-B3-gate 误拦截 Top1 斜率放松进入预注册

## 决策摘要

B3 gate 的误拦截归因显示，`gate_blocked_raw` 中 8 个后续 10 日 -5% 下探样本全部落在 `top1_slope10<=0.00352` 的半区内。本节点将其提升为默认关闭、实验前预注册候选，但不允许直接修改当前 B3 gate。

## 是什么

当前 B3 gate 的启动确认是：

```text
B3 raw 全弱持续
AND
(median_ret<=0 OR ma20_breadth<=0.40)
```

本节点提出的候选不是替代原 gate，而是在 raw 已经触发但原 gate 被拦截时，额外观察一个窄条件：

```text
原 gate blocked
AND
top1_slope10<=0.00352
```

如果该条件成立，未来实验可测试是否允许 limited2 cap80。

## 相比上个节点

上一节点 [[D20260601-B3-gate离线观察复盘聚焦误拦截与漏报]] 只把历史事件整理成 watchlist。本节点进一步完成了误拦截归因：

- non raw 漏报没有形成足够干净的单变量候选。
- active 误伤趋势延续也没有形成稳定候选。
- 只有 `gate_blocked_raw` 中的 Top1 短斜率弱化形成了可预注册的窄假设。

## 当前判断

```text
observe_pre_backtest
```

当前可做什么：

- 建立默认关闭规则规格。
- 做配置审计和 dry-run。
- 做 2024 压力测试和四段历史压力测试。

当前不能做什么：

- 不能直接改当前主候选 B3 gate。
- 不能直接打开 MiniQMT 实盘开关。
- 不能把该样本内阈值解释为样本外有效。
- 不能叠加 R026 作为交易开关。

## 当时问题

B3 gate 四段通过，但历史观察复盘发现 44 个 `gate_blocked_raw` 中有 8 天后续 10 日出现 -5% 下探。问题是这些误拦截是否有共同的、当日可见的解释变量。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 候选覆盖 | `top1_slope10<=0.00352` 命中 22 个 blocked raw，其中包含 8/8 个 false negative |
| 精度提升 | precision 36.36%，高于 blocked raw 基准 18.18% |
| 分段覆盖 | 2022-2023、2024、2025-20260519 的 false negative recall 均为 100% |
| 机制方向 | false negative 的 Top1 短斜率均值 0.00015，非 false negative 均值 0.00366，符合“领涨资产近端失速”解释 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 样本太小 | false negative 只有 8 天 |
| false positive 仍多 | 候选命中 22 天，其中 14 天不是后续 -5% 下探 |
| 无真正样本外 | 阈值来自四段历史未来标签归因 |
| 2020-2021 没有正样本 | 强趋势分段中该条件可能只是额外扰动 |
| 仍未验证交易路径 | 当前只解释未来标签，尚未证明 cap80 后能改善净值路径 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| Top1 斜率弱化是真实误拦截修复条件 | 默认关闭回测应改善 blocked false negative 附近谷底，且收益代价可控 |
| 只是下跌后的滞后变量 | 新增动作只在回撤已深后触发，不能改善谷底 |
| 阈值过拟合 | 分段表现不稳定，尤其 2020-2021 或未来样本拖累收益 |
| 原 gate 保守是正确的 | 放松后 false positive 导致收益下降大于回撤改善 |

## 决策

```text
observe_pre_backtest
```

决策边界：

- 允许进入默认关闭规则规格和实验前预注册。
- 不允许直接进入实盘候选。
- 即使历史四段回测改善，也最多标记为 `observe`，需要未来样本或模拟观察。
