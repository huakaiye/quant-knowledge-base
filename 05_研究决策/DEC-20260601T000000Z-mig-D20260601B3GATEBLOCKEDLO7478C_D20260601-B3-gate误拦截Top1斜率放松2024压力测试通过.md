# D20260601-B3-gate 误拦截 Top1 斜率放松 2024 压力测试通过

## 决策摘要

Top1 低斜率放松在 2024 配对压力测试中没有破坏 B3 gate，收益和回撤均小幅改善，因此允许进入四段历史压力测试；但阈值来自历史未来标签归因，本节点不改变当前主候选，也不允许实盘开启。

## 是什么

本节点测试的是一个窄放松条件：

```text
B3 raw 已触发
AND 原 median/breadth gate blocked
AND top1_slope10<=0.00352
```

若成立，则允许沿用 B3 gate 的 limited2 cap80 动作。它不是新的主防守模块，也不是 R026 叠加规则。

## 相比上个节点

上一节点 [[D20260601-B3-gate误拦截Top1斜率放松进入预注册]] 只允许建立默认关闭规格和首段压力测试。本节点完成了 2024 配置审计、WSL dry-run、正式回测和结果审计，把后续实验顺序从“先跑 2024”改为“补齐三段并做四段压力验证”。

## 当前判断

```text
observe_split_validation
```

当前可做什么：

- 补齐 2020-2021、2022-2023、2025-20260519 三段配置。
- 每段先做配置等价性审计和 WSL dry-run。
- 顺序运行分段压力测试，并审计新增 relax 动作日。

当前不能做什么：

- 不能替代当前 `A2-slope004 + B3-gate-median-or-breadth-cap80` 主候选。
- 不能直接打开 MiniQMT 实盘开关。
- 不能与 R026 叠加为交易 gate。
- 不能把 2024 单段小幅改善解释为样本外有效。

## 当时问题

预注册时只知道 `top1_slope10<=0.00352` 能解释历史 blocked false negative，但尚未证明它能改善真实回测路径。2024 是首段压力测试，用来检查已知 2024-08 误拦截是否被修复，以及 false positive 是否侵蚀收益。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 配置审计通过 | `results/v2/research/R20260601-028/config_audit/summary.json` 显示 `all_passed=true`，未混入 R026、R010D/E、A1/A3/A4 |
| 2024 配对收益小幅改善 | 测试组总收益 +62.4955%，基准 B3 gate +62.0246%，差值 +0.4709pp |
| 回撤没有恶化 | 测试组最大回撤 -29.9477%，基准 -30.0021%，改善 +0.0544pp |
| 交易笔数未增加 | 测试组 267 笔，基准 270 笔，差值 -3 |
| 真正交易影响集中 | relax pass 32 天，但真正 active 只有 3 天，且均来自 baseline blocked raw |
| 2024-08 被覆盖 | 2024-08-01、2024-08-02 两个原 gate blocked raw 日被打开并 active |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 仍是样本内阈值 | `0.00352` 来自历史未来标签归因，不是样本外发现 |
| 交易影响样本太小 | 2024 真正 active 只有 3 天，不能稳定估计收益/回撤性价比 |
| gate 标记变化较多 | relax pass 32 天，说明其他分段可能出现更多非交易或潜在扰动 |
| 2020-2021 没有正样本 | 强趋势段可能只增加噪音，需要优先验证 |
| 改善幅度有限 | 2024 收益和回撤改善都很小，不能直接构成采纳证据 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| Top1 低斜率确实修复 B3 gate 误拦截 | 四段中新增 active 日继续集中在 blocked false negative 附近，且收益不被明显侵蚀 |
| 2024 改善来自偶然交易路径 | 其他分段收益拖累或回撤无改善，尤其 2020-2021 表现差 |
| 低斜率只是滞后风险标签 | 动作日发生在谷底之后，不能改善 T+ 路径 |
| 原 gate 保守仍正确 | 放松导致 false positive 变成真实仓位扰动，收益下降超过回撤改善 |

## 决策

```text
observe_split_validation
```

决策边界：

- 2024 单段通过最低压力门槛，允许补齐四段历史压力测试。
- 本节点不改变当前 B3 gate 主候选，不改变实盘默认关闭边界。
- 若四段中两个以上分段收益拖累且无回撤改善，`kill`。
- 若表现混合但动作日仍有风险解释力，`revise` 为观察或日志层。
- 即使四段均改善，也最多 `observe`，必须等待未来样本或模拟观察。
