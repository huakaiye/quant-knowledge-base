# D20260601-R010B3 limited2 低扰动不直接采纳

## 决策摘要

limited2 证明 B3 全弱持续性有回撤时机信息，但收益代价仍高，不能直接替代 B3-cap90。

## 当时问题

持续性 cap80 会改变后续持仓路径。研究尝试用触发日或有限 N 日动作降低路径扰动，其中 limited2 是优先候选。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 回撤改善 | limited2 相对 B3-cap90 明显改善最大回撤 |
| 时机质量 | active 日相对 cap90 的 log edge 为正，且多落在回撤加深日 |
| 低扰动优于持续状态机 | 比持续切 cap80 更少改变路径 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 收益低于 cap90 | 四段复合收益不如 B3-cap90 |
| 2020-2021 与 2025-2026 拖累 | 非 2024 分段存在收益代价 |
| 不能作为成熟动作载体 | 需要进一步确认启动条件和恢复退出 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| B3 全弱信号有价值，但动作太宽 | 使用启动确认 gate 过滤 raw 噪音 |
| 收益拖累来自路径扰动 | 比较 active 日与非 active 日边际 |

## 决策

```text
revise
```

决策边界：

- limited2 不直接采纳。
- B3 全弱持续性保留为风险时机信号。
- 下一步不继续简单加大防守仓位。

## 下一步

- 预注册并回测 `median_ret<=0 OR ma20_breadth<=0.40` 启动确认 gate。
- R026 暂不扩大交易权限，只观察其是否能做确认层或恢复层。
