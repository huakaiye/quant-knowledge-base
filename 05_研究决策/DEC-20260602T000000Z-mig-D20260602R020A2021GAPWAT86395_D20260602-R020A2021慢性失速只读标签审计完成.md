# D20260602-R020A 2021慢性失速只读标签审计完成

## 决策摘要

R020A 完成 2021-05 触发缺口的只读候选标签审计。综合标签能提前解释该窗口，但历史后 20 日表现不支持交易化，因此决策为 `observe_readonly_explanation_no_trade_edge`。

## 是什么

本节点从 R020/B3 覆盖率缺口复盘接续，只研究 2021-05 这个未解释触发缺口。审计对象包括权益慢性失血、Top1 表面强但斜率退化、ma20 广度临界或快速收缩、波动扩张等只读标签。

平台脚本：

```text
scripts/research/analyze_r020_2021_gap_watch_labels.py
```

输出目录：

```text
results/v2/research/R020/2021_gap_watch_label_audit/
```

本节点不启动回测、不访问数据库、不改实盘配置。

## 相比上个节点

上一节点 [[D20260602-R020-B3覆盖率缺口复盘完成]] 的结论是：2021-05 暂 `park_unresolved_trigger_gap`，若继续只能做只读预注册，不能拼后验仓位规则。

本节点执行了这个受限下一步：把 2021-05 的慢性失速解释做成可复核标签，并用全样本触发率和后 20 日表现审计其噪音。结果没有推翻“不交易化”的边界。

## 当前判断

```text
observe_readonly_explanation_no_trade_edge
```

允许：

- 保留 `r020_2021_gap_composite_watch` 作为 2021-05 缺口解释标签。
- 在后续文档中用该标签描述“权益已失血但 B3 gate 还没触发”的状态。
- 若继续，先做更多独立深回撤窗口的只读审计。

禁止：

- 不把该标签并入 B3 gate。
- 不新增 cap70、cap60 或清仓动作。
- 不修改 MiniQMT 默认开关或 shadow-only 边界。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 综合标签首次命中 2021-05-07 | 比 2021-05-13 谷底提前 4 个交易日 |
| 综合标签 T-10 命中 4 天 | 能覆盖 2021-05 的一部分提前状态 |
| 综合标签全样本触发 98 天 | 触发率 6.36%，不是 R026 宽信号式的大面积覆盖 |
| `equity_drawdown_pressure` 首次命中 2021-05-06 | 权益深回撤叠加近 10 日失血是该窗口最早的解释线索 |
| `breadth_gate_near_miss` 首次命中 2021-05-10 | ma20_breadth 尚未跌破 B3 gate，但已经进入 0.40 到 0.50 临界区 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 综合标签后 20 日平均收益 +3.21% | 仍为正，不能直接支持降仓 |
| 综合标签后 20 日平均最大回撤 -3.31% | 浅于非信号日的 -4.82%，没有更深风险证据 |
| 综合标签后 20 日亏损率 29.59% | 低于非信号日 36.01%，不支持交易型防守 |
| `volatility_expansion_watch` T-10 命中 0 天 | 风险更像谷底确认，不解决提前触发 |
| 2021 样本只有一个关键缺口 | 单点窗口仍有后验解释风险 |

## 竞争性解释

| 解释 | 当前证据 |
| --- | --- |
| 2021-05 是权益慢性失血缺口 | 有证据，综合标签能提前命中 |
| 2021-05 是可交易风险标签 | 证据不足，后 20 日收益和回撤不支持 |
| 2021-05 应通过放宽 B3 gate 解决 | 不支持，既有宽信号历史噪音已经偏大 |
| 2021-05 是单点后验 | 仍可能成立，因此只能观察 |

## 决策

```text
observe_readonly_explanation_no_trade_edge
```

路线边界：

- R020A 从 `park` 推进到“有解释标签但无交易边际”。
- 该标签只能作为研究解释字段，不进入仓位动作。
- 当前主线仍保持 A2-slope004+B3-gate-median-or-breadth-cap80；tiered-v2 仍只允许默认关闭或 shadow-only 观察。

```text

相比上个节点：不再只 park 2021 缺口，而是完成受限只读审计；但没有新增交易触发器。
当前判断：observe_readonly_explanation_no_trade_edge，只保留解释标签，不改 B3 gate 或 tiered-v2。
```
