# D20260601-R010B3 gate 成为当前主候选

## 决策摘要

[[B3 gate]]-median-or-breadth-cap80 四段通过，成为当前 ETF 双池防守研究中最强候选模块。

## 当时问题

limited2 说明 B3 全弱持续性有时机信息，但 raw 全弱信号噪音仍多。需要确认哪些全弱日真的值得切到 cap80。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 四段复合收益 | [[B3 gate]] 复合收益 +1351.04%，优于 [[A2-slope004]]、B3-cap90、B3-cap80 和 limited2 |
| 回撤改善 | 最大回撤约 -30.00%，明显优于 A2 与 B3-cap90 |
| gate 拦截 raw | 拦截 44/122 个 raw 全弱日，说明有效过滤噪音 |
| 有效 cap80 日稀疏 | 只在 66 天生效，不宜继续简单加大防守仓位 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 回撤略弱于 cap80/limited2 | 不是最强防守上界 |
| 仍非实盘默认 | 需要观察日志质量、模拟盘或人工复核 |
| 样本仍有限 | 需要后续持续观察是否跨周期稳定 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| [[median_ret（中位收益）]]/[[ma20_breadth（广度）]] 只是样本内筛选 | 后续做默认关闭观察和新样本复核 |
| 改善来自减少交易扰动 | 对比 raw、gate blocked、active 三类日期 |

## 决策

```text
promote_candidate
```

决策边界：

- A2-slope004 仍是进攻增强主线。
- B3 gate 是风险确认层，不替代趋势跟随。
- B3 gate 已具备默认关闭实盘观察条件，但不直接开启。

## 下一步

- 制定 B3 gate 观察方案，重点看 raw/gate/active 日志和 cap80 生效日。
- R026 只作为解释层和风险复核标记，不直接驱动仓位。
