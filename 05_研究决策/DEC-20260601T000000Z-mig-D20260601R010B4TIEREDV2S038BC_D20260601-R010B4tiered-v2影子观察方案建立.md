# D20260601-R010B4 tiered-v2影子观察方案建立

## 决策摘要

R010-B4 tiered-v2 已补齐默认关闭影子观察方案。当前判断为 `observe_shadow_only`：允许在用户明确授权的模拟盘、小资金或演练日志中记录 strong/extreme 候选等级，但不允许实际把 B3 gate cap80 改成 cap70/cap60。

## 是什么

本节点新增一个只观察、不交易的边界：

```text
ENABLE_B3_GATE_TIERED_V2_SHADOW=0
```

推荐观察组合仅限授权演练：

```text
ENABLE_B3_GATE_MOB_CAP80=1
ENABLE_B3_GATE_TIERED_V2=0
ENABLE_B3_GATE_TIERED_V2_SHADOW=1
```

此时 MiniQMT 会输出：

```text
tiered_level
tiered_candidate_cap
tiered_effective_cap
tiered_reason
```

但 `tiered_effective_cap` 仍保持 B3 gate 的 80%，不因 shadow 降到 70% 或 60%。

## 相比上个节点

上一节点：[[D20260601-R010B4-tiered-v2默认关闭映射完成]]

上一节点只证明实盘文件可表达 tiered-v2 且默认关闭。本节点进一步固定观察方案：即使需要看实时日志，也先用 shadow-only 方式收集证据，不把候选 cap70/cap60 变成实盘动作。

## 当前判断

```text
observe_shadow_only
```

允许：
- 保留 `ENABLE_B3_GATE_TIERED_V2_SHADOW=0` 的默认关闭影子开关。
- 在用户明确授权后，用 shadow-only 方式观察 `normal / strong / extreme`。
- 记录候选 cap、实际 cap、分层原因和后验表现。

禁止：
- 默认开启 `ENABLE_B3_GATE_TIERED_V2`。
- 把 shadow 观察解释为交易上线。
- 绕过 `ENABLE_B3_GATE_MOB_CAP80` 或 B3 gate active 独立评估 tiered-v2。
- 在没有观察样本、人工复核和新决策卡片前 promote。

## 证据

```text

```

新增默认关闭影子开关：

```text
ENABLE_B3_GATE_TIERED_V2_SHADOW=0
```

审计脚本：

```text
scripts/research/audit_r010b4_tiered_v2_live_mapping.py
```

审计输出：

```text
results/v2/research/R010-B4/tiered_v2_live_mapping_audit/
```

审计摘要：

```text
decision=tiered_v2_live_mapped_default_closed_shadow_ready
total_checks=17
passed_count=17
fail_count=0
direct_live_enable=false
```

观察方案：

```text
[[R010B4_tiered-v2默认关闭影子观察方案]]
```

## 反证

| 反证 | 说明 |
| --- | --- |
| 仍然没有真实未来样本 | 当前只完成静态审计和观察方案，未产生 shadow 观察事件 |
| 影子观察依赖 B3 gate 打开 | 若没有用户授权开启 B3 gate 演练，tiered-v2 没有可观察上下文 |
| 历史回测可能高估强防守稳定性 | tiered-v2 四段表现好，但实盘日志仍需验证触发解释和退出行为 |
| strong/extreme 样本可能过少 | 若未来样本不足，不能用少量事件 promote |

## 决策

```text
observe_shadow_only
```

解释：tiered-v2 可以进入默认关闭影子观察，但不能进入实盘默认开启，也不能直接替代 B3-gate-cap80。

## 下一步

1. 若用户明确授权演练观察，按 [[R010B4_tiered-v2默认关闭影子观察方案]] 收集日志。
2. 观察时必须确认 `tiered_enabled=False`、`tiered_shadow=True`、`tiered_effective_cap=80%`。
3. 累积足够 B3 gate active 与 strong/extreme 事件后，再新建决策卡片判断是否进入模拟盘或小资金验证。
