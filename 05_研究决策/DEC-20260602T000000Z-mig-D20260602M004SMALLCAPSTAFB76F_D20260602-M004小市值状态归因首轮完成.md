# D20260602-M004 小市值状态归因首轮完成

## 决策摘要

M004 小市值 T0 首轮只读状态归因完成。当前判断为 `observe_small_cap_input_useful_state_evidence_weak`：小市值继续保留为非 ETF 多策略输入候选，但 R010-A 状态标签对小市值解释力偏弱，不能直接进入自动路由或权重切换。下一步应新建小盘专用状态标签预注册。

## 是什么

本节点读取小市值 T0 四段净值、ETF 主候选拼接净值和 R010-A 状态日志，审计小市值在不同市场状态下的收益、回撤、坏日率、相对 ETF 主候选超额、同跌率和关键窗口表现。

```

```

## 相比上个节点

上一节点：[[D20260602-M004-小市值状态归因预注册完成]]

上一节点只固定了归因问题和只读边界。本节点完成实际归因，并把路线从“用 R010-A 状态解释小市值”修订为“保留小市值输入价值，但另建小盘专用状态标签”。

## 当前判断

```text
observe_small_cap_input_useful_state_evidence_weak
```

当前可做什么：

- 保留小市值 T0 为非 ETF 输入候选。
- 基于低相关和关键窗口互补性，继续研究组合层输入价值。
- 新建小盘专用状态标签预注册，审计小盘相对强弱、流动性压力、质量风险和拥挤踩踏指标。

当前不能做什么：

- 不能用 R010-A `confirmed_state` / `suggested_mode` 直接控制小市值开关。
- 不能进入自动策略路由或权重切换。
- 不能修改实盘默认开关。
- 不能把 2024 低暴露窗口当作可交易防守机制。

## 当时问题

R038 已证明小市值四段净值可用，但小市值是否适合和 ETF 双池共用 R010-A 状态层未知。本轮要判断 R010-A 状态是否足够解释小市值收益，并检查小市值相对 ETF 主候选是否有互补。

## 支持证据

| 证据 | 结果 |
| --- | --- |
| 输入审计 | 1542 个状态日、1542 个小市值交易日，日期缺失 0，`signal_date <= trade_date` 通过 |
| 小市值整体 | 样本日 1541，复合收益 +1704.56%，最大回撤 -16.66%，胜率 51.33% |
| 相对 ETF | 相对 tiered-v2 在 `neutral` 状态累计 log excess +48.15%，小市值回撤 -12.35% vs tiered-v2 -21.60% |
| 低相关 | 与 tiered-v2 日收益相关 0.1240，同日 -1% 坏日率 4.67% |
| 关键窗口 | 2021-05、2024-01、2024-06至09、2024-11至12、2025-04 均相对 tiered-v2 更稳 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| R010-A 状态解释力弱 | `confirmed_state` 日均收益 spread 仅约 0.0477pp |
| 正超额状态不稳定 | 相对 tiered-v2 只有 `neutral` 明确正超额，`favorable` 和 `unfavorable` 为负 |
| 2024 低暴露不可外推 | 2024-06 至 2024-09 小市值收益约 0，更多像低交易/低暴露，不是已验证防守信号 |
| 同跌尾部仍存在 | 与 tiered-v2 同日 -1% 坏日率 4.67%，不能当作天然防守资产 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| 小市值常开优于状态路由 | 先保留输入候选，不用 R010-A 状态开关；后续做组合层消融 |
| R010-A 是 ETF 状态，不适合小盘 | 新建小盘专用状态标签，使用小盘相对强弱、流动性、质量和拥挤指标 |
| 关键窗口优势来自低暴露 | 审计持仓、成交、空仓或低换手机制，不能直接写成防守能力 |

## 决策

```text
observe_small_cap_input_useful_state_evidence_weak
```

决策边界：

- 小市值 T0 保留为非 ETF 多策略输入候选。
- R010-A 状态层暂不作为小市值路由开关。
- 下一步转向小盘专用状态标签预注册。
- 不进入实盘、不改配置、不做自动权重。
