~~~markdown
# D20260602-M004 小市值时点审计通过进入四段预注册

## 决策摘要

M004 小市值 T0 的执行时点与字段可见性审计通过。当前判断从 `small_cap_2024_smoke_pass_needs_timing_audit` 推进为 `small_cap_timing_audit_pass_ready_for_split_prereg`：可以准备四段长测预注册和分段配置，但仍不能进入多策略状态归因、自动路由、权重切换或实盘默认开关修改。

## 是什么

本节点审计小市值 T0 2024 smoke 中最关键的执行时点问题：

```text
prepare_stock_list 09:00
dapan 09:30
daily_adjustment 09:30
handle_T every_bar
```

审计重点是 09:30 同时刻事件顺序、开盘调仓字段可见性、`handle_T` 盘中调度语义、日线与财务字段是否回退到前一交易日，以及分钟数据配置是否支持动态池。

## 相比上个节点

上一节点：[[D20260602-M004-小市值2024Smoke通过需时点审计]]

上一节点只证明 2024 单段 smoke 可生成净值、成交、持仓和报告，但把 09:30 调仓与 `handle_T` 调度列为四段前阻塞。本节点补齐这项审计，并修复平台 V2 同一分钟事件排序语义，使策略源码中的 `dapan -> daily_adjustment` 注册顺序在回测与 live runner 中保持一致。

## 当前判断

```text
small_cap_timing_audit_pass_ready_for_split_prereg
```

允许：

- 新建四段长测预注册或补齐现有研究记录中的四段实验前预测。
- 准备 2020-2021、2022-2023、2024、2025-20260519 分段配置。
- 为四段结果准备净值质量、交易日志、状态对齐和未来函数复核脚本。

禁止：

- 不能把 2024 smoke 外推为 2020-2026 全周期可比净值已补齐。
- 不能直接进入多策略状态归因、自动路由或权重切换。
- 不能修改实盘默认开关。
- 不能跳过四段预注册和结果审计。

small_cap_timing_audit_pass_ready_for_split_prereg
```

解释：执行时点和字段可见性不再阻塞四段长测准备，但当前仍是 `observe`，不是 `promote`。下一步只能进入四段长测预注册，不能直接把小市值纳入状态归因输入层或交易路由。

3. 按平台规则在 WSL 内执行分段回测。
4. 回测完成后审计净值覆盖、成交日志、分钟数据覆盖、状态日期对齐和分段稳定性。
5. 只有四段结果与审计均通过，才允许进入非 ETF 输入层状态归因。
~~~
