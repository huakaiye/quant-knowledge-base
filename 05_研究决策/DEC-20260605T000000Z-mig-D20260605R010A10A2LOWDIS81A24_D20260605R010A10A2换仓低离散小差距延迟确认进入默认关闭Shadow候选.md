# D20260605 R010A10 A2 换仓低离散小差距延迟确认进入默认关闭 Shadow 候选

## 决策

```text
promote_default_off_shadow_candidate
```

允许将 R010-A10S1 预注册为 default-off shadow 候选：

```text
当 A2 发生 switch，
且 dispersion < 0.04，
且 top1_top5_gap < 0.05，
记录 3 日延迟确认 shadow。
```

不允许直接交易化，不允许实盘开启，不允许改变真实 targets、weights、orders。

## 依据

R010-A10 只读代理覆盖：

```text
switch_events = 726
segments = 4
```

全样本机械延迟为负：

```text
3d all_switches sum_delay_edge = -1.059458
5d all_switches sum_delay_edge = -1.129418
```

说明不能通用延迟 A2 换仓。

低离散小差距子集：

```text
dispersion < 0.04
AND top1_top5_gap < 0.05

events = 51
```

3 日延迟代理：

```text
sum_delay_edge = +0.276556
mean_delay_edge = +0.005423
median_delay_edge = +0.005118
delay_help_rate = 0.6078
segments = 4正0负
```

5 日延迟代理：

```text
sum_delay_edge = +0.347383
mean_delay_edge = +0.006811
median_delay_edge = +0.005093
delay_help_rate = 0.6078
segments = 4正0负
```

优先选择 3 日延迟，因为路径扰动小于 5 日，更适合作为第一版 shadow 观察规格。

## 允许

- 实现 R010-A10S1 default-off shadow 日志。
- 先跑 2024 smoke 和四分段隔离审计。
- 仅在隔离通过后讨论 apply smoke。

## 禁止

- 禁止直接改 A2 真实交易路径。
- 禁止直接进入 guarded-live 或实盘。
- 禁止改写真实 targets、weights、orders。
- 禁止扩展小市值、A0/A1/all_actions。

## 下一步

```text
R010-A10S1 shadow 字段至少记录：
  trigger
  delay_days = 3
  old_target
  new_target
  dispersion
  top1_top5_gap
  release_date / release_trade_index
  reason

验收：
  default-off shadow 相对 A2 baseline 权益和交易路径必须 0 差异。
```
