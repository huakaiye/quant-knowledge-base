~~~markdown
# D20260601-R010B4 tiered-v2四段验证通过

## 决策摘要

R010-B4 tiered-v2 已完成默认关闭实现、配置审计、WSL dry-run、2024 smoke 和四段真实路径验证。结果满足预注册 promote 门槛：相对 B3-gate-cap80，四段拼接复合收益提高 +117.30pp，最大回撤改善 +2.88pp，动作日胜率 77.27%。因此本节点将 tiered-v2 晋级为 `promote_candidate_default_closed`。

该判断只允许进入默认关闭候选规格和实盘映射审计，不允许直接上线，不允许替换当前实盘默认开关。

## 是什么

[[tiered-v2（分层强度防守）]] 是 [[B3 gate]] active 后的强度分层规则：

```text
普通 gate：
  B3 gate active，但不满足 strong_risk
  -> cap80

strong_risk：
  B3 gate active
  AND r010b3_all_weak_cum_days >= 10
  AND r010b3_all_weak_persistence_gate_median_ret <= 0
  AND r010b3_top5_filter_exhausted_count >= 4
  -> cap70

extreme_risk：
  strong_risk
  AND (
    r010b3_all_weak_persistence_gate_median_ret <= -0.015
    OR r010b3_all_weak_persistence_gate_ma20_breadth <= 0.25
  )
  -> cap60
```

## 相比上个节点

上一节点：[[D20260601-R010B4-tiered-v2进入预注册]]

上一节点只允许默认关闭实现和 2024 smoke 前置审计。本节点已经完成真实路径验证，并把动作日重建从“实验前证据”降级为辅助解释；最终判断以四段真实回测和审计输出为准。

## 当前判断

```text
promote_candidate_default_closed
```

含义：
- 进入默认关闭候选规格。
- 进入实盘映射、参数默认值、日志字段和回归审计。
- 不直接开启实盘。
- 不替换当前 B3-gate-cap80 主候选边界。
- 不修改 MiniQMT 默认开关。

```text
~~~
