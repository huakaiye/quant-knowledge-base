# D20260602-M005 小盘专用状态标签预注册完成

## 决策摘要

M005 小盘专用状态标签预注册完成。路线从“R010-A ETF 状态无法直接解释小市值”推进到“先审计小盘自身状态标签的输入可得性、时点和覆盖率”，当前不允许回测、路由、权重切换或实盘默认开关修改。

## 是什么

本节点为小市值 T0 建立专用状态标签的实验前边界。候选标签族包括：

- [[小盘相对强弱]]：判断小盘是否相对宽基、ETF 主候选或大盘转弱。
- [[小盘流动性压力]]：判断小票是否出现成交萎缩、跌停扩散或可交易性下降。
- [[小盘质量风险]]：判断小市值失效是否来自盈利质量、壳价值退潮、ST/退市风险或风险重估。
- [[小盘拥挤踩踏]]：判断小盘交易是否过热后集中出逃。

本节点不生成标签结果，只授权下一步只读输入可得性审计。

## 相比上个节点

上一节点：[[D20260602-M004-小市值状态归因首轮完成]]

上一节点证明小市值输入有低相关互补价值，但 R010-A 状态标签对小市值解释力偏弱。本节点不再继续复用 ETF 状态，而是预注册小盘自身状态标签，并固定“先审计输入，不交易化”的边界。

## 当前判断

```text
small_cap_specific_state_label_preregistered_ready_for_input_audit
```

当前可做什么：

- 新增只读输入可得性审计脚本。
- 扫描小盘相对强弱、流动性压力、质量风险和拥挤踩踏四类候选输入。
- 输出字段覆盖率、时点语义、日期对齐和保留/剔除原因。

当前不能做什么：

- 不能启动新回测。
- 不能自动更新数据。
- 不能修改策略、配置或实盘文件。
- 不能生成自动权重、路由规则或交易信号。
- 不能用未来收益、未来回撤或事后谷底反推状态标签。

## 当时问题

R039 显示小市值 T0 与 ETF 主候选低相关，存在组合输入价值；但 R010-A 状态标签只来自 ETF 双池，不能直接解释小市值收益和尾部风险。下一步需要判断小盘自身状态标签是否具备可审计输入。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| R039 输入价值 | 小市值全周期复合收益 +1704.56%、最大回撤 -16.66%，与 tiered-v2 日收益相关 0.1240 |
| R039 互补窗口 | 2021-05、2024-01、2024-06至09、2024-11至12、2025-04 均相对 tiered-v2 更稳 |
| R039 反向证据 | R010-A `confirmed_state` 日均收益 spread 约 0.0477pp，解释力偏弱 |
| 研究信念 | BLF004 要求区分风格、流动性、质量和拥挤，而不是直接用 ETF 状态路由小市值 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 输入可得性未知 | 小盘质量风险、拥挤踩踏和流动性字段未完成全周期覆盖率审计 |
| 时点风险 | 成分、ST、涨跌停、基本面和成交容量字段可能存在 T 日事后可见性问题 |
| 低暴露解释未拆开 | 2024 部分窗口的小市值稳定可能来自低交易/低暴露，不一定来自状态识别 |
| 组合层仍未预注册 | 即使标签可用，也只能进入只读归因，不能直接进入权重路由 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| 小市值常开即可 | 输入审计和后续标签归因只判断状态解释力；若解释弱，则保留常开候选但不路由 |
| 风格强弱足够，质量和拥挤不必要 | 输入审计后按标签族分别归因，不提前删掉质量和拥挤维度 |
| 关键窗口优势来自低暴露 | 后续必须把持仓、成交、现金或低动作状态纳入解释，不把低暴露写成防守信号 |

## 决策

```text
small_cap_specific_state_label_preregistered_ready_for_input_audit
```

决策边界：

- 本节点只改变后续实验顺序：先做小盘专用状态标签输入审计。
- 不改变当前 A2/B3/tiered-v2 主线。
- 不改变小市值 T0 策略本身。
- 不进入组合层自动路由。
