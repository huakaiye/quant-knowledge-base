# D20260603-R010B5R5R3 低MedianRet条件只读复核完成

## 决策摘要

R010-B5R5R3 证明低 `median_ret` 条件有事件级观察价值，但由于分段覆盖不足，不能进入 R010-B5R6、真实路径回测或实盘。

## 是什么

R010-B5R5R3 是 R010-B5 非线性回撤预算路线中的低 `median_ret` 条件复核节点。它不再尝试整体替代 tiered-v2 的固定 cap70/cap60 强度表达，而是只问一个更窄的问题：

当 B3/tiered 已经触发防守，且市场中位收益显著转弱时，非线性建议降仓是否比实际 tiered-v2 仓位更有短期局部收益优势。

本节点使用 R010-B5R5R2 新四段 shadow 样本中的 27 个非零 `cap_delta` 事件，固定两个既有阈值：

- `median_ret <= -0.015`
- `median_ret <= -0.01`

同时保留全量事件、非低 `median_ret` 与 normal-only 作为负控制，并把 `median_ret<=-0.0141` 标记为后验参考、禁止决策。

## 相比上个节点

上一节点：[[D20260603-R010B5R5R2-新样本复核与低MedianRet观察]]

R5R2 解决的是字段阻塞：确认新 shadow payload 带环境字段，且交易隔离通过。但 R5/R5R1 同路径结论仍未反转，低 `median_ret` 只是环境归因线索。

R5R3 改变的是验证方式：把线索变为固定候选，禁止阈值扫描，输出负控制和分段统计。结果显示线索有边际，但分段证据不够。

## 当前判断

```text
observe_low_median_ret_preregistered_candidate_segment_insufficient_no_live
```

当前可做什么：

- 保留低 `median_ret<=-0.015/-0.01` 作为观察级线索。
- 用本轮脚本作为后续新增样本的只读复核口径。
- 在研究库中记录该条件的证据、反证和禁止项。

当前不能做什么：

- 不能进入 R010-B5R6。
- 不能修改 MiniQMT 或实盘默认开关。
- 不能把低 `median_ret` 写成真实路径交易规则。
- 不能采用 `median_ret<=-0.0141` 后验阈值。
- 不能把事件级 close-to-close counterfactual 当成完整回测。

## 当时问题

R010-B5R5 已经反证整体替代固定 cap 强度表达：27 个事件 5日与10日累计 log edge 为负。R010-B5R5R1 的 strong+extreme 窄条件也只是观察级。R010-B5R5R2 新样本复核显示低 `median_ret` 子集偏正，但该结论来自环境归因，必须防止后验过拟合。

因此需要一个固定阈值、带负控制、明确禁止后验阈值的只读复核节点。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| `median_ret<=-0.015` | 8 事件，1日 log edge +0.00433，5日 +0.00597，10日 +0.01302；1日/5日正率 75.00%，10日正率 50.00% |
| `median_ret<=-0.01` | 13 事件，1日 log edge +0.01495，5日 +0.00708，10日 +0.01495；1日/5日正率 69.23%，10日正率 53.85% |
| gate median 字段一致性 | `r010b3_all_weak_persistence_gate_median_ret` 的同阈值结果与 `median_ret` 一致，字段口径没有冲突 |
| 负控制确认失败 | 全量事件 5日 -0.02470、10日 -0.02890；`median_ret>-0.01` 10日 -0.04385；normal-only 10日 -0.03297 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 分段覆盖不足 | 两个主候选只覆盖 2022-2023 与 2024，没有覆盖 2020-2021 与 2025-20260519 |
| 2024 贡献偏重 | `median_ret<=-0.01` 的 2024 10日 log edge +0.02133，而 2022-2023 为 -0.00638 |
| 样本数小 | 主候选仅 8 与 13 个事件，不能承担上线证据 |
| 局部反事实限制 | 本轮只比较事件后的 close-to-close 局部收益，不含成本、冲击、真实路径再平衡和完整组合交易影响 |
| 后验参考更好但禁止使用 | `median_ret<=-0.0141` 10日 log edge +0.02062，但来自 R5R2 后验扫描，不能采用 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| 低 `median_ret` 只是剔除 normal 噪音 | 等新增样本后看 normal/strong/extreme 内部同阈值是否仍稳定 |
| 2024 特定窗口驱动 | 新样本或独立窗口必须覆盖至少 3 个分段 |
| 复述 tiered-v2 强风险层 | 后续需要区分 `median_ret` 与 tiered_level 的交互，而不是把二者混为新规则 |
| 局部反事实与真实交易路径不一致 | 只有在独立预注册真实路径回测中验证后，才可讨论策略化 |

## 决策

```text
observe_low_median_ret_preregistered_candidate_segment_insufficient_no_live
```

决策边界：

- 该节点是观察，不是 promote。
- 该节点不是 R010-B5R6。
- 该节点不是实盘变更。
- 低 `median_ret` 条件只能作为后续新增样本复核口径。
- `median_ret<=-0.0141` 只能作为后验禁止参考，不得进入规则。
