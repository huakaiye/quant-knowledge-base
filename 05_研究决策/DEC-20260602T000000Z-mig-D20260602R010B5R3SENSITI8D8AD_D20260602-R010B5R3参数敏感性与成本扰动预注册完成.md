# D20260602-R010B5R3 参数敏感性与成本扰动预注册完成

## 决策摘要

R010-B5R2 已升级为离线候选，但还不能实盘，也不能说参数稳健。R010-B5R3 的决策是：

```text
sensitivity_cost_preregistered_no_live
```

含义：

```text
只允许按预注册的 6 个单变量参数扰动和 2 个成本扰动推进。
先跑 2024 smoke，通过后再四段。
不得大规模扫描，不得实盘。
```

## 是什么

R010-B5R3 不寻找新最优参数，而是验证 R010-B5R2 主候选：

```text
nl_endo_gate_any_d10_a5_f70
```

是否对相邻扰动稳健。

参数扰动：

| 变体 | 改动 |
| --- | --- |
| `d08_a5_f70` | 阈值 10% -> 8% |
| `d12_a5_f70` | 阈值 10% -> 12% |
| `d10_a3_f70` | 衰减 5 -> 3 |
| `d10_a7_f70` | 衰减 5 -> 7 |
| `d10_a5_f65` | 最低缩放 70% -> 65% |
| `d10_a5_f75` | 最低缩放 70% -> 75% |

成本扰动：

| 变体 | 含义 |
| --- | --- |
| `comm2x` | 佣金提升到 0.02% |
| `comm2x_slip2bps` | 佣金 0.02% + 2bps 滑点 |

## 相比上个节点改变了什么

上个节点结论是 `promote_offline_candidate_no_live`，证明非线性叠加版四段真实路径有效。

本节点新增：

- 把“是否参数偶然”列为必须验证的问题。
- 把成本扰动纳入候选资格，而不是只看原始低成本回测。
- 明确禁止后验扩大参数网格。
- 明确本轮不测试替代 B3 gate 或替代 tiered-v2。

## 当前判断

```text
sensitivity_cost_preregistered_no_live
```

可以做：

- 运行 2024 smoke。
- 2024 通过后，运行四段敏感性和成本扰动。
- 生成敏感性汇总、成本压力汇总和动作日审计。

不能做：

- 不能新增参数组合。
- 不能根据 2024 结果反向修改阈值。
- 不能修改 MiniQMT。
- 不能实盘。
- 不能宣称替代 B3 gate 或 tiered-v2。
