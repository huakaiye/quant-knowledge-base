本节点判断 [[R20260604-060_R010A4R1_进攻软排序默认关闭实现审计]] 之后的 `v1_signal_gate` 是否可以进入 2025/2026 样本外。

| 证据 | 结论 |
| --- | --- |
| 资金曲线 | baseline 与 v1 逐日完全一致 |
| 收益/回撤 | 总收益、年化、最大回撤、最终权益全部 0 差异 |
| 交易 | 两组均为 281 笔 |

## 决定

```text
revise_gate_or_scoring_before_out_of_sample
```

v1 不能进入样本外。它不是实现未生效，而是动作未能改变选标。

允许：

- 做 R010-A4S3 近失效归因。
- 围绕 Top1 豁免、soft_rank_scale、差值型加分或 raw_top1_gap 分层设计 v2。

禁止：

- 不跑 2025/2026 样本外。
- 不把 v1_signal_gate 当作收益优化候选。
- 不上线、不修改实盘。

## 下一节点

下一节点应是：

```text

```

若 v2 仍不能在 2024 smoke 中产生可解释选标差异，R010-A4 软排序方向应转入只读或 kill。
