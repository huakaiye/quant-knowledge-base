# D20260601-R026B-light-cap90 首段反证

## 决策摘要

R026B-light-cap90 的 2024 配对首段没有通过最低收益/回撤交换门槛：相对同配置 B3 gate 控制组收益下降 1.08pp，最大回撤只改善 0.036pp。因此该分支不继续升级为仓位动作候选，暂不进入四段长测。

## 是什么

本节点审计的是 `R20260601-027` 的 2024 配对回测：

- 控制组：A2-slope004 + B3-gate-median-or-breadth-cap80，同样 `warmup_days=260`，关闭 R026B-light。
- 实验组：在控制组基础上开启 `r026_b_light_cap_enabled=true`，只用 `signal_B_equity_defense`，B3 gate inactive 时触发 1 日 cap90。

审计输出：

```text
results/v2/research/R20260601-027/result_audit/
```

## 相比上个节点

上一节点 [[D20260601-R026B-light-cap90进入预注册]] 只允许低权限离线验证，尚未产生回测证据。本节点新增：

- 默认关闭策略参数已经实现。
- 四段实验配置已完成审计。
- 2024 配对控制组已补跑，排除了 `warmup_days` 差异。
- 触发日志已确认 R026B-light 只新增 3 次 cap90 叠加。

关键变化是：问题从“能不能表达该实验”变成“首段证据是否值得继续长测”。答案是否定的。

## 当前判断

```text
revise
```

具体边界：

- 不把 R026B-light-cap90 升级为候选仓位动作。
- 暂不继续四段正式长测，除非先重写假设为“日志复核/确认层”而不是“自动 cap90 动作”。
- R026 的定位回到低覆盖风险标记、触发解释、恢复或退出确认审计。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 收益拖累 | 2024 控制组 +62.02%，R026B-light +60.94%，差 -1.08pp |
| 回撤改善不足 | 控制组最大回撤 -30.00%，R026B-light -29.97%，只改善约 0.036pp |
| 触发太少 | 242 条动作日志中，B层信号 9 天，实际 light active 仅 3 天 |
| 增量动作有限 | 新增 3 条 ACTION_APPLIED，主要把 10% 风险仓转为防守 ETF |
| 配对控制排除混杂 | 同样 `warmup_days=260` 的控制组复现 B3 gate 2024 +62.02% | 

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 只完成 2024 单段 | 尚不能证明四段一定失败 |
| B层信号仍较干净 | `signal_B_equity_defense` 历史精度仍高于宽信号 |
| 2024 触发簇少 | 单段结果可能受 2024 年少数日期路径影响 |

这些反对证据不足以支持继续直接长测，因为本实验预注册目标是“不破坏 B3 gate 的收益/回撤性价比”。2024 首段已经显示收益代价远大于回撤收益。

| R026B-light-cap90 | +60.94% | -29.97% | 160941.72 | 270 |

R026B-light 生效信号日为 `2024-01-04`、`2024-01-26`、`2024-02-22`，对应交易日为 `2024-01-05`、`2024-01-29`、`2024-02-23`。

## 决策

```text
revise
```

R026B-light-cap90 不进入下一轮交易候选。若后续继续 R026，应改为：

- gate active / gate blocked / non-raw drawdown 三类日期上的解释力审计。
- B层信号对误触发、漏触发和恢复退出的辅助判断。
- 默认关闭日志字段或人工复核字段，而不是自动 cap90 仓位动作。
