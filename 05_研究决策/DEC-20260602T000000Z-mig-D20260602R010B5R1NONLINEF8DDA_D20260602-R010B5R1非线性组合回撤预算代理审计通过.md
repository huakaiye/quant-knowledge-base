# D20260602-R010B5R1 非线性组合回撤预算代理审计通过

## 决策摘要

R010-B5R1 把上一轮“机械硬分档降仓”修订为“条件化非线性连续缩放”。净值层代理审计显示，最强 endogenous 版本在收益保留率 107.73%、最大回撤改善 +1.14pp、动作日胜率 81.48% 的条件下通过代理门槛。

当前决策为：

```text
promote_proxy_candidate_for_true_path_no_live
```

这只表示可以进入真实策略路径回测预注册，不表示可以实盘。

## 是什么

R010-B5R1 是组合层二级风控研究。它不替代 A2、[[B3 gate]] 或 [[tiered-v2（分层强度防守）]]，而是在 tiered-v2 已经给出 normal、strong 或 extreme 风险动作的日期，额外用[[组合回撤预算]]决定是否连续缩放风险仓。

预注册公式：

```text
risk_scale = max(floor, exp(-decay * max(drawdown_abs - threshold, 0)))
```

本轮主候选：

```text
nl_endo_gate_any_d10_a5_f70
账户自身回撤驱动
B3/tiered normal/strong/extreme 日允许缩放
10% 回撤后指数衰减
最低 70%
```

## 相比上个节点改变了什么

上个节点 R010-B5 的结论是 `revise_proxy_return_drag_no_live`，原因是全周期硬分档会长期低仓锁定。

本节点做了三处收敛：

- 从全周期触发改为只在 B3/tiered 风险窗口内触发。
- 从硬阈值 cap 分档改为指数连续缩放。
- 从数百到上千个动作日压缩到 27 个动作日。

## 当前判断

```text
promote_proxy_candidate_for_true_path_no_live
```

可以做：

- 新建 R010-B5R2 真实策略路径回测预注册。
- 研究如何在正式路径中表达 `nl_endo_gate_any_d10_a5_f70`。
- 审计交易成本、防守 ETF、现金路径和影子账本必要性。

不能做：

- 不能修改 MiniQMT。
- 不能上线。
- 不能把代理结果当作正式回测结果。
- 不能继续扩展大规模非线性参数扫描。
