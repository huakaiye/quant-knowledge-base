# D20260601-R010B3 cap90 进入组合前审计

## 决策摘要

B3-cap90 从日志/风险雷达升级为 A2-slope004 组合前验证候选，但不是实盘上线结论。

## 当时问题

R010-B3 的全弱信号是否能在不显著损害收益的情况下改善 A2-slope004 的回撤路径。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 四段复合收益 | B3-cap90 相对 A2-slope004 收益为正 |
| 最大回撤改善 | 相对 A2-slope004 回撤改善约 2.58pp |
| 动作日贡献 | B3 触发日相对 A2 有正向 log edge |
| 成本压力 | 额外 5bp/10bp 成本后仍保持正向边际 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 最深窗口未解决 | 2024-06 到 2024-09 最大残余窗口仍存在 |
| cap80 回撤更好 | 更强防守可进一步改善回撤，但收益代价更高 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| 改善来自降低风险仓位，不是识别风险 | 比较触发日、非触发日和分段边际 |
| 2024 贡献过大 | 四段分段验证和成本审计 |

## 决策

```text
promote_to_combination_pre_audit
```

决策边界：

- 只进入组合前审计，不直接上线。
- 必须默认关闭，且保持日志可审计。
- 不混入 A1/A3/A4、R026 硬过滤或直接清仓。

## 下一步

- 固化 A2-slope004+B3-cap90 组合规格。
- 做实盘参数映射、日志字段和交易冲击审计。
