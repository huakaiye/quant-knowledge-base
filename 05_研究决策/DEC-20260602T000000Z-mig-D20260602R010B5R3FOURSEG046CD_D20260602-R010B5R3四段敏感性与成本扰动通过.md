# D20260602-R010B5R3 四段敏感性与成本扰动通过

## 决策摘要

R010-B5R3 四段参数敏感性与成本扰动压力验证通过，允许进入 R010-B5R4 默认关闭影子观察边界设计。

当前判断：

```text
promote_to_r010b5r4_shadow_observation_no_live
```

## 是什么

本节点验证非线性组合回撤预算在 B3 gate/tiered-v2 已识别风险窗口后，作为二级仓位预算叠加项是否具备参数与成本稳健性。

它只评估：

```text
在既有 B3 gate 与 tiered-v2 防守框架内，
非线性回撤预算能否稳定改善仓位强度表达。
```

它不评估：

```text
是否可以替代 B3 gate 风险窗口识别。
是否可以替代 tiered-v2 strong/extreme 识别。
是否可以直接进入实盘。
```

## 相比上个节点

上一节点：[[D20260602-R010B5R3-部分替代边界定义]]

上个节点只是定义“部分替代”的研究边界。本节点新增四段实证结果，并把路线从“等待四段压力验证”推进到“允许设计影子观察”。

## 当前判断

```text
promote_to_r010b5r4_shadow_observation_no_live
```

当前可做：

- 设计 R010-B5R4 默认关闭影子观察。
- 记录非线性建议 cap 与实盘 tiered cap 的差异。
- 继续审计重复 run 和结果可复现性。

当前不能做：

- 不能修改实盘默认开关。
- 不能用 R010-B5R3 替代 B3 gate。
- 不能用 R010-B5R3 替代 tiered-v2 strong/extreme 识别。
- 不能因为 `d10_a5_f65` 表现更好而后验换参。

## 当时问题

R010-B5R2 叠加版四段表现较好，但可能来自单点参数偶然性或成本假象。因此 R010-B5R3 预注册了 6 个相邻参数扰动和 2 个成本扰动，要求先完成四段验证再讨论后续路线。

## 支持证据

| 证据 | 说明 |
| --- | --- |
| 数据完整性 | 40/40 配置有结果，0 缺失，0 个非零退出但有结果 |
| 参数邻域 | 6/6 相邻扰动均优于 B3-gate-tiered-v2，回撤改善约 +1.63pp 至 +2.13pp |
| 成本扰动 | `comm2x` 与 `comm2x_slip2bps` 下候选相对同成本基准仍保留收益和回撤优势 |
| 新样本拖累 | 2025-20260519 最大收益拖累为 -2.80pp，未超过 -5pp 验收线 |
| 交易扰动 | 成交笔数变化很小，未出现异常放大 |

## 反对证据

| 证据 | 说明 |
| --- | --- |
| 重复 run | 5 个配置存在重复 summary，已在 `sensitivity_four_segment_duplicate_runs.csv` 中记录，后续引用必须同时保留该证据风险 |
| 2021-05 漏报 | 非线性回撤预算不能解决风险窗口识别漏报，只能在 gate 已触发后调仓 |
| replacement 反证 | R010-B5R2 replacement 版本已反证“替代风险强度识别”的路径 |
| 后验换参风险 | `d10_a5_f65` 四段最好，但这只能证明邻域稳健，不能改主参数 |

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| 结果来自 2024 单段贡献 | 四段拼接与 2025-20260519 拖累检查均已执行，仍需后续影子观察继续验证 |
| 结果来自成本假象 | 已用 same-cost baseline 对比，候选在两个成本扰动下仍占优 |
| 结果来自重复 run 选择 | 汇总脚本已改为记录所有重复 run，并输出被选中 job；后续需要保留 duplicate_runs 审计文件 |

## 决策

```text
promote_to_r010b5r4_shadow_observation_no_live
```

决策边界：

- R010-B5R3 不直接实盘。
- R010-B5R3 不直接顶替原有防御模块。
- 下一步只允许设计 R010-B5R4 默认关闭影子观察。
- 若 R010-B5R4 影子观察显示建议 cap 稳定且误伤可控，再预注册 R010-B5R5 部分替代固定 cap70/cap60 的强度表达。
