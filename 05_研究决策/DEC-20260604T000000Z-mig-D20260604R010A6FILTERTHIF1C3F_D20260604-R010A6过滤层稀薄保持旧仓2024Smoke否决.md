# D20260604-R010A6 过滤层稀薄保持旧仓2024Smoke否决

## 决策摘要

R010-A6 严格版 `filter_all_weak -> 保持旧仓` 已完成默认关闭实现、shadow 观察和 2024 apply smoke。

结论：不通过。

保持旧仓并没有兑现 A5 的 episode 代理收益，2024 样本内真实路径收益和回撤均劣于 A2-slope004 baseline。因此该规则不进入 OOS，不进入实盘候选，保持默认关闭。

## 证据

shadow 观察：

| 指标 | 数值 |
| --- | ---: |
| shadow日志行 | 242 |
| 触发次数 | 9 |
| 触发动作 | A2 = 9 |
| 真实订单差异 | 0 |
| 收益/回撤/交易笔数差异 | 0 |

apply smoke：

| 指标 | baseline | A6 strict apply | 差异 |
| --- | ---: | ---: | ---: |

| 年末权益 | 156104.68 | 152510.86 | -3593.82 |

权益路径差异：

```text

```

## 当前判断

```text
reject_trade_promotion_keep_default_off
```

允许：

- 保留默认关闭代码、配置、审计脚本和 shadow 解析脚本。
- 使用 9 个触发窗口做只读失败归因。
- 继续研究“新目标差时应该选择什么替代路径”，但必须重新预注册。

不允许：

- 不跑年度 OOS。
- 不扩大到 A1/A0 或 all_actions。
- 不把 `filter_all_weak` 直接映射为保持旧仓规则。
- 不进入 MiniQMT、实盘开关或 shadow-only 实盘观察。

## 关键解释

A5 的代理口径捕捉的是“换到新标的后的 episode 收益为负”，但真实交易问题是：

```text
不换到新标的时，旧仓是否更好？
```

2024 apply smoke 给出的答案是否定的。部分窗口中，旧仓比新目标更差，尤其 5月、7月和11月触发后，保持旧仓扩大了相对损失。

## 下一步

若继续推进 R010-A，只能先做只读失败归因：

```text
对9个触发窗口比较 原A2目标 / 旧仓 / B0目标 三条路径，寻找可事前分离标签。
```

在找到能解释真实替代路径优劣的事前标签前，不再新增保持旧仓类交易化测试。
