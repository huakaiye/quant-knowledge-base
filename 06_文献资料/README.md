# 文献资料说明

文献资料用于保存论文和外部研究的中文文献卡。文献卡不是全文翻译，而是服务策略研究的结构化摘要。

## 使用方式

1. 发现论文线索，先放 `00_待处理/`。
2. 读完摘要，创建 `LIT` 文献卡。
3. 提炼出因子、数据、机制或模块灵感后，在 `07_因子数据灵感/` 创建对应卡片。
4. 进入实验后，在实验记录中引用文献卡和灵感卡 ID。

## 目录说明

| 目录 | 说明 |
| --- | --- |
| `00_待处理/` | 未读或只登记线索 |
| `01_经济学顶刊/` | 经济学顶刊论文 |
| `02_金融学顶刊/` | 金融学顶刊论文 |
| `03_资产定价/` | 因子和风险溢价 |
| `04_市场微观结构/` | 流动性、交易成本、订单行为 |
| `05_行为金融/` | 投资者行为、注意力、情绪 |
| `06_宏观金融/` | 宏观状态、利率、通胀、信用 |
| `07_计量与机器学习/` | 统计方法、预测模型、稳健性 |
| `08_已归档/` | 已处理或暂不使用 |

详细规则见 `08_方法论/文献归档规范.md`。
