# 台账说明

台账是研究库的结构化索引，用于快速检索和避免资料散落。台账不替代正文，正文负责解释研究逻辑和证据。

## 台账列表

| 文件 | 用途 |
| --- | --- |
| `研究方向台账.csv` | 所有策略级方向和模块级子方向 |
| `实验台账.csv` | 所有实验记录、平台配置和结果路径 |
| `决策台账.csv` | 所有路线决策 |
| `文献台账.csv` | 所有文献卡 |
| `策略资产台账.csv` | 策略档案和平台资产 |
| `术语台账.csv` | 术语词条索引 |

## 使用规则

1. 新增核心文档时，同步新增台账行。
2. 修改状态时，同步更新台账状态。
3. 台账中的路径写相对路径，平台路径写绝对路径。
4. 大型结果不复制进研究库，只登记路径。
5. 台账允许被脚本生成或检查，但最终内容要能人工阅读。
