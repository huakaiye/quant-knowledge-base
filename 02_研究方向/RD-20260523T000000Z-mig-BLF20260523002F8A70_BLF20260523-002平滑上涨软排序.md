# BLF20260523-002 平滑上涨软排序

## 信念陈述

平滑上涨质量不适合作为硬准入门槛，但可能适合作为动量排序的条件性质量调节项。它应当在中等广度、中等离散、主题轮动仍在分化的环境中使用，而不是在全面强上涨或极端爆发主线中常开。

## 机制解释

平滑上涨描述的是趋势路径质量。路径更平滑的上涨可能更少依赖单日脉冲，后续延续性更好。但硬过滤会误伤刚启动的强趋势；常开软排序又会在全面上涨或极端主线中压慢追涨速度。因此它更像条件性质量权重，而不是独立交易信号。

## 适用边界

| 维度 | 当前判断 |
| --- | --- |
| 适用资产 | ETF 动量候选池 |
| 适用频率 | 日频选标或排序加权 |
| 适用市场状态 | 中等广度、中等离散、主线尚未完全扩散 |
| 可能失效环境 | 全面强上涨、政策脉冲、Top1 极强趋势、低离散噪声 |

## 可观察预测

如果该信念成立，应当观察到：

- 硬过滤版本在样本外不稳定，尤其会错过新启动强趋势。
- 软排序版本优于硬过滤，且交易次数不显著增加。
- 在中等广度和中等离散环境中，软排序相对原始动量有正主动收益。
- 在全面强上涨和 Top1 极强环境中，软排序可能拖累原始动量。
- 加入 Top1 极强豁免或状态门控后，R9 的负主动收益月份应减少。

## 竞争性解释

| 解释 | 如何区分 |
| --- | --- |
| R9 改善来自少数极端月份 | 去掉最佳月份、按年份滚动、按月份贡献拆分 |
| 平滑质量只是低波动暴露 | 加入波动率分组或波动率中性化诊断 |
| 软排序公式参数偶然有效 | 做小范围参数扰动，不做大规模寻优 |
| 市场状态门控才是核心，平滑质量边际贡献很低 | 对比有无 R9 的同一状态门控版本 |

## 证据记录

### 支持证据

- [[R20260521-009_平滑上涨软排序实验]] 显示软排序在 2022-2024 合并区间优于基准和硬过滤。
- [[R20260522-010_ETF双池轮动适用市场环境归因]] 显示 R9 在中等环境中更有效。

### 反对证据

- R009 在 2025 单年退化，说明常开软排序并不稳健。
- R008 显示硬过滤样本外表现不够强，不应直接作为准入规则。

### 尚未解决的问题

- 平滑质量应当作用于排序、仓位还是换手控制。
- Top1 极强豁免条件如何定义，避免事后调参。
- R9 与市场状态门控的边际贡献如何分离。
